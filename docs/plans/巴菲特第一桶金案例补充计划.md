## 核心原则

1. **不保留原始电子书内容** - 删除 `wiki/ebooks/` 目录，不推送远程
2. **提取原创研究** - 从每个案例中提炼投资逻辑、关键数据、经验教训
3. **案例研究独立成页** - 每个案例单独一页，便于多来源融合
4. **公司页面引用案例** - 公司主页面简洁，链接到详细案例
5. **明确引用来源** - 所有案例研究注明来自《巴菲特的第一桶金》

## 执行步骤

### 单案例执行流程

1. **读取原始案例** - 从 `wiki/ebooks/巴菲特的第一桶金/caseXX-案例名.md` 提取内容
2. **创建案例研究页面** - 按标准结构写入 `wiki/research/cases/公司名-年份-主题.md`
3. **更新公司页面** - 更新或创建 `wiki/companies/公司名.md`，添加案例链接
4. **检查交叉链接** - 确保相关概念、人物、信件页面链接正确
5. **标记完成** - 更新本计划文档中的状态
