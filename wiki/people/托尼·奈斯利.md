# 托尼·奈斯利

## 人物简介

托尼·奈斯利（Tony Nicely，1944-2019）是GEICO保险公司的传奇CEO，在GEICO工作超过50年，带领公司从地区性小保险公司发展成为全美第二大汽车保险公司。

### 基本信息

- **出生年份**：1944年
- **逝世年份**：2019年
- **职业**：保险业管理者
- **最著名任职**：GEICO CEO（1993-2018）
- **GEICO工龄**：50年以上

### 职业轨迹

1. **18岁加入GEICO**，从基层做起
2. **1993年升任CEO**，接替前任
3. **2018年退休**，结束了在GEICO的职业生涯
4. **在任期间**，将GEICO市场份额从2.0%提升至7.7%

## 与巴菲特的关系

### 巴菲特的高度评价

巴菲特在多封致股东信中盛赞托尼·奈斯利的管理才能：

#### 1996年信

> "去年我们完成对盖可保险的全资收购时，我们的期望很高——如今这些期望不但全部实现，而且被一一超越。不论从商业角度还是个人角度都是如此：盖可保险的运营主管托尼·奈斯利(Tony Nicely)是一位卓越的企业管理者，与他共事也是一件愉快的事。"

#### 1998年信

> "好的理念加上好的经理人，你就一定能获得好的结果。这种完美组合在盖可保险身上得到了最好的体现。理念是低成本的汽车保险，通过直销客户来实现；经理人则是托尼·奈斯利(Tony Nicely)。说得简单点，整个商业世界没有人能比Tony把盖可保险经营得更好。"

#### 2000年信

> "盖可保险的CEO托尼·奈斯利(Tony Nicely)仍然是每个老板梦寐以求的经营人才。他做的每一件事都恰到好处。"

#### 2015年信

> "1993年托尼成为CEO，此后公司一直高歌猛进。没有比托尼更好的管理者了。在我认识他的40年里，他的每一项决策都恰到好处。"

## 主要成就

### GEICO的蜕变

| 年份 | 市场份额 | 备注 |
|------|---------|------|
| 1993年 | 2.0% | 托尼出任CEO |
| 1996年 | 2.5% | 伯克希尔收购完成 |
| 2003年 | 5.0% | |
| 2007年 | 7.2% | |
| 2016年 | 7.7% | |

### 关键成功因素

1. **坚守承保纪律**：在扩大市场份额的同时保持严格的承保标准
2. **低成本直销模式**：维持GEICO的低成本优势
3. **品牌建设**：大幅增加广告投入，从3100万美元增至7.51亿美元
4. **市场扩张**：在竞争激烈的汽车保险市场持续抢占份额

## 管理风格

### 特点

- **精准的直觉**：对市场变化反应敏锐
- **充沛的精力**：80多岁时仍每天坚持工作
- **完美的执行力**：每一个决策都恰到好处
- **承保纪律**：坚守严格的承保标准

### 巴菲特的评价

> "认识他的人都知道，自从18岁进入公司以来，Tony跟盖可保险已经深深缠绵了41个年头，他的业绩正是这份热情的明证。"

## 退休

2018年，托尼·奈斯利从GEICO CEO职位退休，比尔·罗伯茨（Bill Roberts）接任。
