# 2014 副董事长的思考

> **原文**: [raw/special/2014-副董事长的思考.md](../../raw/special/2014-副董事长的思考.md)
> **执笔**: 查理·芒格
> **场合**: 2014年伯克希尔50周年补充演讲

---

## 分析

### 背景

作为伯克希尔50年的近距离见证者，芒格从副董事长视角补充了巴菲特的演讲，系统阐述了伯克希尔成功的管理体系。

### 伯克希尔系统13条原则

| 序号 | 原则 |
|------|------|
| 1 | 多元化企业集团，回避无法预判的业务 |
| 2 | 子公司享有极大自主权 |
| 3 | 总部几乎空空如也 |
| 4 | 以财产险为核心，产生浮存金 |
| 5 | 无统一人事、激励、退休制度 |
| 6 | 董事长保留少数关键职责 |
| 7 | 现金收购，不用股票 |
| 8 | 不分红，留存收益检验标准 |
| 9 | 合理价格买好生意+优秀CEO |
| 10 | CEO选拔看重诚信、能力、干劲、热爱 |
| 11 | 几乎从不出售子公司 |
| 12 | 极少调动子公司CEO |
| 13 | 不因年龄强迫退休，保持极低债务 |

### 四大成功因素

1. 巴菲特自身的独特优势
2. 伯克希尔系统的独特优势
3. 好运气
4. 股东和仰慕者的热爱

### 继任者评价

> "阿吉特·贾恩和格雷格·阿贝尔都是经过验证的杰出管理者，用'世界级'来形容他们恐怕还说低了。我会选择'世界领先'这个词。"
