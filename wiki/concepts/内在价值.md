# 内在价值

## 定义

内在价值（Intrinsic Value）是巴菲特投资哲学的核心概念，指的是一家企业基于其未来现金流折现值的真实经济价值。它是企业能够为其所有者创造的财富的现值，是投资者应该关注的"本质"价值，而非市场价格所反映的"表面"价值。

## 巴菲特观点

巴菲特多次强调内在价值是评估投资的"唯一合乎逻辑的估值方法"。他引用凯恩斯的话："内在价值应该等于投资者在整个生命周期内能够从资产中提取的现金的折现值。"

### 关键原则

1. **内在价值是估计值**：精确的内在价值无法计算，只能估算
2. **不确定性**：不同投资者对同一企业的内在价值评估可能差异很大
3. **安全边际**：只有当市场价格显著低于内在价值时，才存在投资机会
4. **长期视角**：内在价值随时间和企业经营状况而变化

## 计算方法

虽然巴菲特强调内在价值无法精确计算，但通常采用现金流折现（DCF）方法：

```
内在价值 = Σ [CFt / (1+r)^t] + 终值/(1+r)^n
```

其中：
- CFt = 第 t 年的现金流
- r = 折现率
- t = 年份
