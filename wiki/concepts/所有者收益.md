# 所有者收益 (Owner Earnings)

## 定义

所有者收益是巴菲特提出的衡量企业真实盈利能力的指标。

## 计算公式

```
所有者收益 = 净利润 
           + 折旧、损耗、摊销 
           - 必要的资本支出 
           - 额外的营运资本需求
```

## 与 GAAP 的区别

| GAAP 净利润 | 所有者收益 |
|------------|-----------|
| 包含折旧 | 扣除折旧 |
| 忽略资本支出 | 扣除必要资本支出 |
| 可能高估收益 | 更真实反映 |

## 巴菲特观点

> "这是真正重要的数字——即使你拥有整个企业，这也是你真正能得到的"

## 应用场景

- **评估内在价值**: 重要的估值基础
- **比较企业**: 跨行业比较盈利能力
- **再投资决策**: 评估保留收益的回报率
