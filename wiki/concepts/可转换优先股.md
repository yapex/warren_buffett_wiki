# 可转换优先股

> **攻守兼备的投资工具**：可转换优先股结合了债券的安全性（固定股息+优先清偿权）和股票的上涨潜力（转换为普通股的权利）。

## 定义

可转换优先股是一种特殊类型的优先股，持有者有权在特定条件下将其转换为发行公司的普通股。这种结构同时具备：
- **债券特征**：固定股息、优先清算权
- **股票特征**：转换为普通股、分享上涨收益

## 巴菲特的可转换优先股投资

### 投资特点

巴菲特持有的可转换优先股通常具有以下特点：

| 特征 | 说明 |
|------|------|
| 转换期限 | 在相当长时间内不可转换 |
| 转换价格 | 通常高于购买时的普通股价格 |
| 股息率 | 提供稳定的现金回报 |
| 保护条款 | 优于普通股的清算优先权 |

### 投资逻辑

1. **下行保护**：固定股息提供稳定的现金收益
2. **上行潜力**：保留分享普通股上涨的权利
3. **长期视角**：不允许短期炒作普通股价格

## 可转换优先股 vs 普通优先股

| 维度 | 可转换优先股 | 普通优先股 |
|------|--------------|------------|
| 收益潜力 | 转换后可能大幅上涨 | 受限于固定股息 |
| 风险水平 | 较低（有债券特征） | 中等 |
| 转换权 | 有 | 无 |
| 适合时机 | 看好长期发展 | 追求稳定收益 |
