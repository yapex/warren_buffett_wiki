# 资本配置 (Capital Allocation)

## 定义

资本配置是指企业管理层决定如何分配公司资源（留存收益、债务、股权等）。

## 资本配置选项

| 选项 | 说明 |
|------|------|
| 再投资 | 扩大现有业务 |
| 收购 | 购买其他公司 |
| 分红 | 返还给股东 |
| 回购 | 买回自家股票 |
| 还债 | 降低负债 |

## 优秀的资本配置原则

### 理性决策

- 评估每项投资的预期回报
- 优先考虑回报最高的机会

### 股东利益

- 不为管理层私利浪费资本
- 在股价低估时回购

### 避免愚蠢多元化

- 不为扩张而扩张
- 只做有把握的投资

## 巴菲特的资本配置能力

- **保险浮存金**: 灵活运用低成本资金
- **收购时机**: 在市场恐慌时出击
- **回购决策**: 在股价远低于价值时回购
