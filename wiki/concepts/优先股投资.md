# 优先股投资

> **1989年开始的系统性使用**：巴菲特从1989年开始大规模使用优先股作为投资工具，在危机期间投资优质企业。

## 定义

优先股投资是巴菲特投资工具箱中的重要组成部分。与纯粹的普通股投资不同，优先股投资提供了额外的下行保护——固定的股息支付和优先清算地位，同时保留了转换为普通股的可能性。

## 巴菲特的优先股投资策略

### 投资标准

1. **可转换特性**：能够转换为普通股，参与企业长期增长
2. **稳定现金流**：优先股股息提供持续的现金回报
3. **优质企业**：只投资于具有持久竞争优势的企业
4. **合理价格**：即使有优先股保护，也不支付过高价格

### 投资时机

- **市场恐慌时期**：优质公司的优先股往往被错杀
- **企业危机时刻**：危机不等于致命问题
- **并购套利**：利用并购确定性获利

## 经典案例

### 吉列（1989）

| 项目 | 详情 |
|------|------|
| 投资时间 | 1989年 |
| 投资方式 | 优先股投资 |
| 投资金额 | 6亿美元 |
| 结果 | 巨大回报，成功转换为普通股 |

### 全美航空 US Airways（1989）

| 项目 | 详情 |
|------|------|
| 投资时间 | 1989年 |
| 投资方式 | 优先股投资 |
| 结果 | **失败案例**——航空业超出能力圈 |

这是巴菲特少数失败的优先股投资之一，证明即使工具正确，用在错误的行业也会失败。

### 盖可保险（1976）

| 项目 | 数据 |
|------|------|
| 首次买入时间 | 1976年7月 |
| 普通股投资 | 411万美元，129万股 |
| 优先股投资 | 1942万美元，198万股（1976年11月） |
| 1976年总投资 | 约2353万美元 |
| 后续追加 | 至4700万美元（1980年） |
| 持股比例 | 约33%（1980年） |

## 优先股 vs 普通股的风险对比

| 维度 | 优先股 | 普通股 |
|------|--------|--------|
| 收益稳定性 | 固定股息 | 波动较大 |
| 下行保护 | 优先分配权 | 无 |
| 上行潜力 | 受限于转换条款 | 理论上无限 |
| 投票权 | 通常无 | 通常有 |

## 教训总结

1. **优质企业是前提**：优先股结构再好也无法挽救烂生意
2. **能力圈仍然适用**：即使投资工具匹配，行业判断错误仍会导致失败
3. **长期持有的力量**：吉列案例证明，长期持有优质优先股可获得巨大回报
4. **不要迷信工具**：USAir案例提醒，工具正确不等于投资正确
