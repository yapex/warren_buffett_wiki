# 1956 年有限合伙协议

> **原文**: [raw/partnership/zh/1956-有限合伙协议.md](../../raw/partnership/zh/1956-有限合伙协议.md)
> **签署人**: 沃伦·E·巴菲特

---

## 分析

### 背景

- **成立日期**: 1956年5月1日
- **合伙企业名称**: 巴菲特联合有限合伙公司 (Buffett Associates, Ltd.)
- **存续期限**: 1956年5月1日至1976年4月30日
- **普通合伙人**: 沃伦·E·巴菲特

### 有限合伙人

| 合伙人 | 关系 |
|--------|------|
| 小查尔斯·E·彼得森 | 巴菲特室友、邻居 |
| 伊丽莎白·B·彼得森 | 查尔斯·彼得森之母 |
| 多丽丝·B·伍德 | 巴菲特之姐 |
| 杜鲁门·S·伍德 | 巴菲特姐夫 |
| 小丹尼尔·J·莫内恩 | 巴菲特发小、律师 |
| 威廉·H·汤普森 | 巴菲特岳父 |
| 爱丽丝·R·巴菲特 | 巴菲特姑妈 |

### 出资额

总出资额: **105,000 美元**

| 合伙人 | 出资额 |
|--------|--------|
| 爱丽丝·R·巴菲特 | $35,000 |
| 威廉·H·汤普森 | $25,000 |
| 伊丽莎白·B·彼得森 | $25,000 |
| 其他四位 | 各$5,000 |

### 利润分配

- **利息**: 各有限合伙人按资本账户余额获得 4% 年利率
- **利润分成比例**:
  - 巴菲特姑妈 (Alice): 7/42
  - 岳父 (Thompson): 5/42
  - 彼得森夫人 (Peterson): 5/42
  - 其他四位: 各 1/42

### 关键条款

- 有限合伙人无权转让出资地位
- 不得接纳新有限合伙人
- 普通合伙人去世、退休或丧失行为能力时，合伙人无权继续经营
