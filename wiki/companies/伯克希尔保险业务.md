# 伯克希尔保险业务

> 浮存金模式的起点，伯克希尔帝国的核心引擎

## 公司简介

伯克希尔保险业务是伯克希尔·哈撒韦的核心业务板块，1967 年收购国民赔偿保险后正式起步，通过保险浮存金模式成为伯克希尔投资的资金来源。

## 伯克希尔保险业务的发展

### 时间线

| 年份 | 事件 |
|------|------|
| 1967 | 收购国民赔偿保险，保险业务起步 |
| 1969 | 收购更多保险公司，扩大浮存金规模 |
| 1970s-1980s | 保险浮存金持续增长 |
| 1990s-2000s | 通用再保险、伯克希尔能源等加入 |

### 投资概要

| 维度 | 数据 |
|------|------|
| 起点 | 1967 年国民赔偿保险 |
| 核心模式 | 保险浮存金再投资 |
| 关键人物 | 杰克·林沃尔特（国民赔偿保险创始人） |
| 浮存金规模 | 从 1967 年的数千万到 2020s 的 1400 亿 + 美元 |

伯克希尔保险业务是伯克希尔帝国的核心：
