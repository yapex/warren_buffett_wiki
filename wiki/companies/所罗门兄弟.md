# 所罗门兄弟

> 巴菲特投行危机处理的经典案例，临危受命出任临时董事长的惊险之旅

## 公司简介

所罗门兄弟（Salomon Brothers）曾是全球最大的投资银行之一，1980 年代在并购热潮和债券市场快速发展中赚得盆满钵满。1997 年被旅行家集团（Travelers Group）收购。

## 巴菲特与所罗门兄弟的关系

### 时间线

| 年份 | 事件 |
|------|------|
| 1976 | 首次认识 CEO 约翰·古夫兰（盖可保险危机时） |
| 1987 | 投资 7 亿美元购买 9% 可转换优先股 |
| 1991.8 | 国债拍卖丑闻爆发，巴菲特出任临时董事长 |
| 1992.6 | 卸任临时董事长，回归伯克希尔 |
| 1997 | 所罗门被旅行家集团收购，伯克希尔获得丰厚回报 |

### 投资概要

| 维度 | 数据 |
|------|------|
| 投资时间 | 1987 年 |
| 投资金额 | 7 亿美元 |
| 投资形式 | 可转换优先股 |
| 股息率 | 9% |
| 转换价格 | 38 美元/股 |
| 赎回条件 | 1995-1999 年间分期赎回 |
| 最终结果 | "远好于预期"（1997 年被收购） |
| 持有周期 | 10 年（1987-1997） |
