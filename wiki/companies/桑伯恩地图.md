# 桑伯恩地图

> 格雷厄姆式净流动资产价值投资的经典案例，巴菲特合伙公司早期的成功投资

## 公司简介

桑伯恩地图公司（Sanborn Map Company）是一家出版地图和保险承保手册的公司，1950 年代因其投资组合价值远超股价而成为格雷厄姆式投资的典型案例。

## 巴菲特与桑伯恩地图的关系

### 时间线

| 年份 | 事件 |
|------|------|
| 1958-1960 | 巴菲特合伙公司大举买入桑伯恩地图股票 |
| 1958 | 股价约 45 美元，仅投资组合价值就达 65 美元/股 |
| 1960 | 推动公司分拆，释放投资价值 |

### 投资概要

| 维度 | 数据 |
|------|------|
| 投资时间 | 1958-1960 年 |
| 买入价格 | 约 45 美元/股 |
| 投资组合价值 | 65 美元/股 |
| 地图业务价值 | 免费赠送 |
| 巴菲特合伙公司持股 | 约 23%（进入董事会） |
| 最终结果 | 推动分拆，释放价值 |

## 核心投资逻辑

- [净流动资产价值](../concepts/净流动资产价值.md) — 股价低于流动资产价值
- [安全边际](../concepts/安全边际.md) — 投资组合价值提供下行保护
- 积极股东主义 — 进入董事会推动价值释放

## 历史意义

桑伯恩地图是巴菲特合伙公司早期的经典投资：
1. 展示了格雷厄姆式"烟蒂投资"的精髓
2. 体现了巴菲特作为积极股东的影响力
3. 为后来的投资风格演变奠定了基础
