# 洛克伍德公司

> 巴菲特在格雷厄姆 - 纽曼公司参与的套利交易，所有权思维的早期体现

## 公司简介

洛克伍德公司（Rockwood & Co.）是一家巧克力豆制造商，1950 年代因持有大量可可豆库存而具有特殊投资价值。当时可可豆价格飙升，公司通过股票换可可豆的方式进行资产重组。

## 巴菲特与洛克伍德公司的关系

### 时间线

| 年份 | 事件 |
|------|------|
| 1956 | 巴菲特在格雷厄姆 - 纽曼公司工作期间参与洛克伍德套利 |
| 1956 | 格雷厄姆选择套利（每股赚 2 美元差价） |
| 1956 | 巴菲特选择长期持有可可豆（最终回报更高） |

### 投资概要

| 维度 | 数据 |
|------|------|
| 投资时间 | 1956 年 |
| 公司类型 | 巧克力豆制造商 |
| 套利方案 | 股票换可可豆（每股换 36 美元可可豆） |
| 市场价格 | 34 美元/股 |
| 格雷厄姆套利空间 | 每股 2 美元 |
| 巴菲特的选择 | 长期持有可可豆而非套利 |
