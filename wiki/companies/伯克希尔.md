# 伯克希尔·哈撒韦 (Berkshire Hathaway)

## 简介

美国跨国控股公司，由沃伦·巴菲特创建并领导，从纺织厂转型为全球最大投资集团之一。

| 项目 | 详情 |
|------|------|
| 成立 | 1839 年（原名 Valley Falls Company） |
| 巴菲特接手 | 1965 年 |
| 总部 | 内布拉斯加州奥马哈 |
| 董事长 | 沃伦·巴菲特 (1965-present) |
| 副董事长 | 查理·芒格 (1978-2023) |

## 发展历程

### 纺织时代 (1965 之前)

- 最初是罗得岛州的纺织厂
- 1965 年前已持续亏损多年
- Seabury Stanton 家族经营

### 巴菲特接管 (1965)

| 事件 | 详情 |
|------|------|
| 收购方式 | 通过合伙企业逐步买入 |
| 控制权 | 获得董事会多数席位 |
| 管理层 | 任命 Kenneth V. Chace |
| 股价表现 | 1965 年涨 49.5% |

### 转型扩张 (1965-1980s)

- 逐步退出纺织业务
- 收购国民保险公司 (1967)
- 涉足保险、糖果、零售

### 投资帝国 (1980s-现在)

- 收购可口可乐 (1988)
- 收购吉列 (1989)
- 收购伯克希尔·哈撒韦再保险 (1998)
- 收购伯灵顿铁路 (2010)
- 持有苹果大量股份

## 业务板块

| 板块 | 代表公司 | 特点 |
|------|----------|------|
| 保险 | 盖可、国家雇员保险、再保险 | 浮存金优势 |
| 铁路 | BNSF | 垄断性资产 |
| 公用事业 | 伯克希尔能源 | 稳定现金流 |
