# 伯克希尔·哈撒韦

> 从纺织厂到投资帝国，巴菲特资本配置的载体

## 公司简介

伯克希尔·哈撒韦公司（Berkshire Hathaway Inc.）原是一家纺织公司，1962-1965 年被巴菲特逐步收购，后来转型为投资控股公司，成为巴菲特资本配置的核心载体。

## 巴菲特与伯克希尔的关系

### 时间线

| 年份 | 事件 |
|------|------|
| 1962 | 巴菲特开始买入伯克希尔股票 |
| 1964 | 与 CEO 斯坦顿的"口头协议"破裂 |
| 1965 | 获得控股权，接管公司 |
| 1967 | 收购国民赔偿保险，开启保险业务 |
| 1970s-1980s | 逐步关闭纺织业务，转型投资 |
| 1980s-2020s | 成为全球最大投资控股公司之一 |

### 投资概要

| 维度 | 数据 |
|------|------|
| 首次买入 | 1962 年 |
| 控股时间 | 1965 年 |
| 收购成本 | 约 1200 万美元（逐步买入） |
| 转型时间 | 1970s 关闭纺织业务 |
| 当前市值 | 数千亿美元（2020s） |

## 巴菲特的反思

> "收购伯克希尔是我犯过的最大错误之一...如果我当初把资金投入其他生意，回报会高得多。"
> 
> —— 巴菲特，1989 年致股东信

但巴菲特也承认，伯克希尔成为了他资本配置的完美载体。

## 历史意义

伯克希尔·哈撒韦是巴菲特投资生涯的核心：
1. 从纺织厂到投资帝国的转型
2. 展示了资本配置的力量
3. 成为巴菲特投资哲学的实践平台
