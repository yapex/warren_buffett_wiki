# 邓普斯特农机

> 巴菲特困境企业 turnaround 的首次尝试，从投资到控股的演变

## 公司简介

邓普斯特农机公司（Dempster Mill Manufacturing）是一家农机制造商，1960 年代初因经营困境被巴菲特合伙公司控股，成为巴菲特首次尝试困境企业扭亏的案例。

## 巴菲特与邓普斯特农机的关系

### 时间线

| 年份 | 事件 |
|------|------|
| 1961 | 巴菲特合伙公司开始买入邓普斯特农机股票 |
| 1961-1963 | 持续增持至控股地位 |
| 1963 | 巴菲特进入董事会，推动管理层更换 |
| 1963 | 聘请哈里·伯特负责运营 |
| 1963 后 | 公司逐步扭亏为盈 |

### 投资概要

| 维度 | 数据 |
|------|------|
| 投资时间 | 1961-1963 年 |
| 投资性质 | 从投资到控股 |
| 控股比例 | 约 70%+ |
| 关键行动 | 更换管理层、削减成本、关闭亏损部门 |
| 最终结果 | 公司扭亏为盈，巴菲特获利退出 |

## 核心投资逻辑

- 困境企业投资 — 买入经营困难但资产扎实的企业
- 控股投资 — 通过控股推动企业变革
- [管理层更换](../concepts/管理层.md) — 聘请专业经理人扭亏

## 历史意义

邓普斯特农机是巴菲特投资生涯的重要里程碑：
1. 首次尝试困境企业控股投资
2. 展示了积极股东主义的力量
3. 为后来的伯克希尔控股模式积累了经验
