# 巴菲特合伙公司

> 巴菲特投资生涯的起点，13 年辉煌业绩的载体

## 公司简介

巴菲特合伙公司（Buffett Partnership Ltd.）是巴菲特 1956 年创立的投资合伙企业，运营至 1969 年解散，期间实现了年化 29.5% 的惊人回报。

## 巴菲特合伙公司的历史

### 时间线

| 年份 | 事件 |
|------|------|
| 1956 | 巴菲特合伙公司成立，初始资金 10.5 万美元 |
| 1956-1969 | 13 年运营期，年化回报 29.5% |
| 1967-1969 | 市场狂热，巴菲特反思并写信给合伙人 |
| 1969 | 合伙公司解散，资产分配给合伙人 |

### 投资业绩

| 维度 | 数据 |
|------|------|
| 运营时间 | 1956-1969 年（13 年） |
| 年化回报 | 29.5% |
| 道指对比 | 道指年化 7.4% |
| 超额回报 | 年化 22.1% |
| 初始资金 | 10.5 万美元 |
| 最终规模 | 约 1 亿美元 |

## 历史意义

巴菲特合伙公司是投资史上的传奇：
1. 13 年年化 29.5% 的惊人业绩
2. 展示了格雷厄姆式投资的威力
3. 为伯克希尔的运营奠定了基础
4. 1969 年解散体现了巴菲特的诚信和原则
