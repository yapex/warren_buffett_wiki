# 威利家具：1995-周日不营业的家具帝国

## 一、决策场景还原

### 时间与市场环境

| 项目 | 内容 |
|------|------|
| 投资时间 | 1995年 |
| 所属行业 | 家具零售 |
| 目标市场 | 犹他州（美国西部内陆州） |
| 市场环境 | 1990年代中期，美国零售业竞争激烈，周日营业已成为行业惯例 |
| 经济背景 | 美国经济持续增长，消费支出旺盛 |

### 公司的背景

| 项目 | 内容 |
|------|------|
| 创立时间 | 1932年 |
| 创始人 | R.C. Willey |
| 起点 | 从皮卡车上卖电器起家 |
| 接手时间 | 1954年 |
| 接班人 | Bill Child（22岁，从岳父手中接手） |
| 核心理念 | 周日不营业（宗教原则/摩门教信仰） |
| 经营哲学 | 基督教/摩门教价值观 |
| 1995年地位 | 犹他州最大家具零售商 |

### 巴菲特的位置

- 1995年，巴菲特已执掌伯克希尔·哈撒韦30余年
- 彼时巴菲特已完成多项重大收购，包括1994年收购GEICO
- Bill Child主动致电巴菲特，表达了出售意向
- 巴菲特在前往棕榈泉的路上顺道参观了威利家具门店

## 二、关键数据

| 项目 | 内容 |
|------|------|
| 收购价格 | 每股$22,000（锁定价格） |
| 收购年份 | 1995年 |
| Bill Child初始投资（1954年） | $35,000贷款起家 |
| 1954年年营业额 | $25万 |
| 1999年年营业额 | $3.42亿（45年增长1368倍） |
| 博伊西分店首年销售 | $3000万（1999年） |
| 拉斯维加斯首年销售 | $8600万（2001年） |
| 员工数量 | 不到2500人（极高人效） |
| 覆盖州数（2018年） | 犹他州、爱达荷州、内华达州、加州，共约10家店 |

## 三、巴菲特的决策过程

### 1995年：信任创始人的直觉

Bill Child致电巴菲特谈及出售事宜。巴菲特立即报价：每股$22,000，锁定价格。然而Bill Child坚持要巴菲特亲自来看门店，他说："这对你不公平。"

**Buffett原话：**
> "坦白说，我非常怀疑'周日不营业'这套做法能否在一个全新的市场站稳脚跟。"

巴菲特去棕榈泉的路上顺道参观了威利家具门店。

### 1999年：致股东信中的评价

在1999年致股东信中，巴菲特详细评价了这笔投资：

**Buffett原话：**
> "Bill Child代表美国最好的一面。"

> "犹他州家具业霸主威利家具(R.C. Willey)，我们在1995年从比尔·查尔德(Bill Child)家族手中买下了这家公司。Bill和他的大部分管理团队都是摩门教徒，因此威利家具的门店周日从来不开门。"

> "遵从你自己的商业判断和宗教信仰吧。"

### 1999年：博伊西分店的成功

Bill Child决定自己买地建房开拓博伊西市场，并承诺如果失败，伯克希尔不用付钱。结果第一年销售就达到$3000万，超过预期目标。

### 2001年：拉斯维加斯扩张

威利家具进入拉斯维加斯市场，第一年销售高达$8600万，再次证明了其商业模式的可复制性——即使在"周日不营业"的原则下。

## 四、决策对比点

### 1. 宗教原则 vs. 商业惯例

| 维度 | 传统商业观点 | 巴菲特的判断 |
|------|------------|-------------|
| 周日营业 | 零售行业标准做法，最大化销售收入 | 尊重管理层宗教信仰，不强制改变 |
| 结果 | 短期收入可能减少 | 长期赢得客户信任，形成差异化优势 |

**分析：** 巴菲特并未将自己的商业判断强加于Bill Child，而是选择了"遵从你自己的商业判断和宗教信仰"——这种授权哲学使Bill Child能够保持核心价值观经营。

### 2. 家族企业 vs. 企业集团

| 维度 | 典型收购（企业集团） | 威利家具模式 |
|------|-------------------|-------------|
| 创始人角色 | 被替换或边缘化 | Bill Child继续执掌 |
| 经营自主权 | 总部管控严格 | 高度自治 |
| 激励机制 | 短期业绩考核 | 长期价值创造 |

**分析：** 巴菲特选择信任Bill Child的管理团队，而非将威利家具纳入标准化管理体系。这反映了巴菲特对"傻子也能经营"哲学的坚持——找到优秀的管理者，然后放手。

### 3. 稳健扩张 vs. 快速复制

| 维度 | 常规扩张策略 | 威利家具的策略 |
|------|-------------|---------------|
| 地理扩张 | 快速复制到多个市场 | 稳扎稳打，验证一个市场再进下一个 |
| 风险承担 | 总部承担扩张风险 | Bill Child个人承担失败损失 |
| 资本配置 | 集团统一调配 | 管理者自掏腰包买地建房 |

**分析：** Bill Child在博伊西分店采用"自己出资、承诺失败不用伯克希尔付钱"的模式，体现了对自身判断的信心和对巴菲特的尊重。

### 4. 短期怀疑 vs. 长期验证

| 维度 | 1995年初期的疑问 | 后续验证结果 |
|------|-----------------|-------------|
| 周日不营业能否盈利 | 巴菲特持怀疑态度 | 持续增长，证明可行 |
| 能否复制到新市场 | 怀疑论者认为不可能 | 博伊西、拉斯维加斯均获成功 |
| 宗教原则是否限制扩张 | 担心人才招聘受限 | 摩门教社区提供稳定劳动力 |

**分析：** 巴菲特承认自己的判断有误——"周日不营业"并未阻碍威利家具的成功，反而成为其品牌特色和差异化竞争优势。

### 5. 资本效率 vs. 规模扩张

| 维度 | 指标 | 威利家具数据 |
|------|------|-------------|
| 员工效率 | 员工数量 | 不到2500人 |
| 营收规模 | 1999年营业额 | $3.42亿 |
| 人均产出 | 估算 | 约$14万+/人 |
| 店铺数量 | 2018年 | 约10家 |

**分析：** 威利家具用极少的人力实现了极高的产出，展现了"砖块积累"式增长的资本回报率优势。

### 6. 主动争取 vs. 被动等待

| 维度 | Bill Child的选择 | 结果 |
|------|-----------------|------|
| 出售决策 | 主动致电巴菲特 | 获得伯克希尔的品牌背书和资本支持 |
| 扩张决策 | 自己出资承担风险 | 赢得巴菲特的深度信任 |
| 继任安排 | 保持摩门教团队稳定 | 企业文化得以延续 |

**分析：** Bill Child不是被动等待伯克希尔的指示，而是主动提出让巴菲特无法拒绝的条件（如失败自己承担损失），最终实现了双赢。

## 五、后续发展

### 时间线

| 年份 | 事件 |
|------|------|
| 1932年 | R.C. Willey从皮卡车上卖电器起家 |
| 1954年 | R.C. Willey因胰腺癌去世，Bill Child（22岁）接手 |
| 1954年 | Bill说服First National Bank of Layton贷款$35,000 |
| 1983年 | 已成为犹他州最大家具零售商 |
| 1995年 | 巴菲特收购威利家具 |
| 1999年 | 博伊西分店开业，第一年销售$3000万 |
| 2001年 | 拉斯维加斯开业，第一年销售$8600万 |
| 2018年 | 犹他州、爱达荷州、内华达州、加州共约10家店 |

### 财务数据

| 年份 | 年营业额 | 增长倍数 |
|------|---------|---------|
| 1954年 | $25万 | 基准 |
| 1999年 | $3.42亿 | 1368倍 |
| 复合增长率 | 约17.5% | 45年 |

### 经验教训

1. **最强业务是砖块积累的：** 威利家具45年增长1368倍，靠的是持续深耕本地市场，而非激进扩张。

2. **做正确的事：** 周日不营业看似损失收入，实际赢得了客户信任和员工忠诚。

3. **学会完全信任某些人：** 巴菲特对Bill Child的充分授权，换来了管理层的全力投入。

4. **资本回报率是关键：** 不到2500人创造$3.42亿营收，人效极高。

5. **从错误中学习：** 巴菲特最初怀疑"周日不营业"能否成功，后来承认自己的判断有误。

6. **保持谦逊：** 即使是巴菲特，也会对自己的偏见进行调整。
