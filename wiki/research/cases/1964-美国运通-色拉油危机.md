# 美国运通：1964年色拉油危机投资案例

> **核心来源**：巴菲特亲笔信（1994年回顾）+《巴菲特的第一桶金》

---

## 一、决策场景还原

### 时间与市场环境

1960年代初，美国运通是华尔街的宠儿，凭借旅行支票和信用卡业务建立了强大的市场地位。作为金融服务公司，声誉是其核心资产。

### 公司危机：色拉油丑闻

1963年9月，一场金融骗局震惊华尔街：

- 一个名叫 Anthony De Angelis 的骗子买下大豆色拉油储存在仓库中
- 将油罐注入海水，只在表面浮一层真正的色拉油
- 骗取美国运通出具虚假库存仓单
- 从51家银行获得数千万美元贷款
- 骗局揭穿时，贷款银行损失超过1.5亿美元

**市场反应**：
- 股价从64美元断崖式跌至38美元（跌幅约40%）
- 华尔街分析师担心公司能否存活
- 投资者恐慌性抛售

### 巴菲特的位置

**年龄与经验**：34岁，管理巴菲特合伙公司已有8年。

**资金规模**：1964年初，合伙公司资金约1700万美元。

**关键决策**：将合伙公司约**40%的资金**投入美国运通——这是合伙公司有史以来最大的一笔投资，仅花费1300万美元，却获得了超过5%的股权。

---

## 二、关键数据

### 投资决策

| 指标 | 数值 |
|------|------|
| **投资时间** | 1964年初至1966年 |
| **投资金额** | 约1300万美元 |
| **买入价格** | 约38-60美元区间 |
| **持股比例** | 超过5% |
| **仓位占比** | 合伙公司资产的40%（最大单笔投资） |
| **卖出时间** | 约1967年 |
| **卖出价格** | 约180美元/股 |

### 投资结果

| 时间点 | 情况 |
|--------|------|
| 1964年买入 | 成本约1300万美元，持股超5% |
| 1967年卖出 | 股价约180美元，获利数倍 |
| 1994年回顾 | "1964年盈利1250万美元，1994年14亿美元" |

## 三、巴菲特的决策过程

### 1964-1966年：建仓过程

**市场调研（Scuttlebutt）**

面对危机，巴菲特没有依赖华尔街分析师的报告，而是亲自进行实地调研：

| 调研对象 | 调研发现 |
|---------|---------|
| 餐厅顾客 | 顾客照常使用美国运通卡付账，完全不受丑闻影响 |
| 商家 | 依然接受美国运通卡 |
| 银行/旅行社 | 人们依然喜欢使用运通的旅行支票 |
| 竞争对手 | 仍认为运通是强有力的对手 |

**核心结论**：美国运通的经济特许权完好无损。尽管它在色拉油事件上栽了跟头，但凭借过去的可信度依然拥有强大的商誉。

### 1994年：长期回顾——"绝佳的投资机会"

> "在其他情况下，当一家优秀的企业遭遇一次性的重大但可解决的难题时——就像多年前美国运通和盖可保险都经历过的那样——**绝佳的投资机会就出现了**。"
> 
> ——巴菲特，1989年致股东信

### 1994年：详细回顾——"合伙公司有史以来最大的一笔投资"

> "我在美国运通的投资历史包含好几段插曲：1960年代中期，该公司刚被臭名昭著的色拉油丑闻重创之后，**我们将巴菲特合伙公司约40%的资金押在这只股票上——这是合伙公司有史以来最大的一笔投资**。我得补充一句，当时这笔投入只花了1,300万美元，却让我们拥有了美国运通超过5%的股权。"
> 
> ——巴菲特，1994年致股东信

### 1994年：关于投资回报——"长期复利的力量"

> "在我写这封信的此刻，我们持有将近10%的股权，成本为13.6亿美元。**美国运通1964年的盈利为1,250万美元，1994年则达到14亿美元。**"
> 
> ——巴菲特，1994年致股东信

### 1997年：关于管理层——高尔夫球场上的决策

> "就在我考虑是否应该在转换时卖出美国运通时，我恰好在缅因州普劳茨内克与赫兹租车的CEO弗兰克·奥尔森一起打高尔夫。Frank是一位才华横溢的管理者，因为业务关系对信用卡行业了如指掌。于是从第一洞开球起我就追着他问个不停。到了第二洞果岭上，Frank已经让我完全相信美国运通的企业卡是一项**绝佳的特许经营生意**——我当场决定不卖了。"
> 
> ——巴菲特，1997年致股东信

### 1989年：关于投资哲学——"躲避恶龙"

> "当然，有时候你不得不啃硬骨头，就像我们当初在布法罗创办周日版报纸那样。在其他情况下，当一家优秀的企业遭遇一次性的重大但可解决的难题时——就像多年前美国运通和盖可保险都经历过的那样——**绝佳的投资机会就出现了**。但总体而言，我们**靠躲避恶龙取得的成绩，远比屠龙要好得多**。"
> 
> ——巴菲特，1989年致股东信

## 四、决策对比点

> 以下问题没有标准答案，供读者思考

### 1. 信息层面
**实际发生**：巴菲特亲自到餐厅、商店、银行、旅行社调研，而非依赖华尔街报告。

**问题**：你是从研报中了解公司，还是亲自去接触实际使用者？这些"接地气"的信息如何改变你的判断？

### 2. 时机层面
**实际发生**：巴菲特在股价暴跌40%后大举买入，仓位高达40%。

**问题**：股价跌多少才算"跌到位"？你是越跌越买，还是等"确认"再动手？

### 3. 仓位层面
**实际发生**：单一投资占总资产的40%，这是巴菲特的常规操作（合伙公司允许单一投资最高40%）。

**问题**：你如何决定单笔投资的仓位？"鸡蛋放在一个篮子里"vs"分散投资"，哪个风险更高？

### 4. 风险层面
**实际发生**：市场认为美国运通可能倒闭。但巴菲特判断，其核心特许权（旅行支票、信用卡）未受损。

**问题**：你如何区分"致命危机"和"可解决的暂时困难"？什么才是公司的"核心竞争优势"？

### 5. 卖出层面
**实际发生**：巴菲特在1967年卖出（股价涨到180美元），获利了结。

**问题**：同样是优秀公司，巴菲特为什么卖出美国运通，却长期持有华盛顿邮报？什么条件下你会卖出？

### 6. 核心判断
**实际发生**：巴菲特说"绝佳的投资机会出现在一家优秀企业遭遇一次性的重大但可解决的难题时"。

**问题**：你是否同意这个判断？这与"不要接飞刀"的投资格言有何冲突？

## 五、后续发展

### 时间线

| 年份 | 事件 |
|------|------|
| 1963年9月 | 色拉油丑闻曝光，股价从64美元跌至38美元 |
| 1964年初 | 巴菲特合伙公司开始建仓美国运通 |
| 1964-1966年 | 持续买入，占合伙公司资产40% |
| 约1967年 | 股价涨至180美元，巴菲特卖出获利 |
| 1991年 | 伯克希尔以3亿美元买入美国运通Percs（可转换优先股） |
| 1994年 | Percs转换为普通股，继续持有约10%股权 |
| 1997年 | 高尔夫球场上，Frank Olson说服巴菲特加仓 |
| 2003年后 | 美国运通成为伯克希尔长期核心持仓之一 |

### 投资方法论的演进

这笔投资代表了巴菲特从"格雷厄姆式"向"品质投资"过渡的关键时刻：

| 维度 | 格雷厄姆标准 | 美国运通 |
|------|-------------|---------|
| 净流动资产 | 强劲资产负债表 | ❌ 不符合 |
| 低市盈率 | 估值便宜 | △ 危机前估值不低 |
| 清盘价值 | 资产折价 | ❌ 不适用 |
| **特许经营权** | — | ✅ 核心判断 |
| **管理层质量** | — | ✅ 声誉即生命 |
