# 内布拉斯加家具城：1983-家族企业收购

> **参考来源**：《巴菲特的伯克希尔崛起》案例 3（格伦·阿诺德著）

---

## 案例概述

1983 年，伯克希尔以约 6000 万美元收购内布拉斯加家具城（NFM）90% 股权。创始人 B 夫人（Rose Blumkin）1917 年从俄罗斯移民，1937 年以 500 美元起步创立 NFM，坚持"物美价廉说实话"的经营理念。这是巴菲特"收购优秀家族企业并让其自主经营"模式的典范。NFM 成为美国家具零售业传奇，持续数十年高速增长。

### 关键数据

| 项目 | 数据 |
|------|------|
| 收购时间 | 1983 年 |
| 收购价格 | 约 6000 万美元（90% 股权） |
| 创始人 | B 夫人（Rose Blumkin） |
| 创立时间 | 1937 年（500 美元起步） |
| 1983 年销售额 | 约 1 亿美元 |
| 利润率 | 约 4-5%（低价策略） |
| 营业面积（1983） | 约 20 万平方英尺 |
| 营业面积（现在） | 168.9 万平方英尺，占地 77 英亩 |
| 商品种类 | 超过 85000 种 |
| B 夫人工作至 | 103 岁 |
| B 夫人去世 | 1998 年（104 岁） |

---

## 投资背景

### B 夫人的传奇故事

**创业历程**：
- 1917 年从俄罗斯移民美国，24 岁，几乎身无分文，不懂英语，从未上过学
- 俄国革命期间通过贿赂边防警卫从白俄罗斯逃出
- 1937 年在地下室开了一家小店，带着四个孩子看着店铺
- 经营座右铭："物美价廉说实话"
- 最困难时甚至卖掉自己家里的家具来还债
- 经过数十年奋斗，NFM 取得巨大成功

**经营哲学**：
> "我不懂书本上的那些理论，我用的就是老办法：说真话，买的对，卖得便宜。"
> — B 夫人

**巴菲特的评价**：
> "如果我开始创业时，可以像在美国橄榄球联盟选拔球员那样，从全国顶尖商学院中挑选 25 名优秀毕业生，或从《财富》500 强公司的 CEO 们中挑选 25 个，或者我可以请 B 夫人来经营公司，那我就请 B 夫人好了，像她这样的人可不多。"

### 家具零售行业特征

**行业特点**：
- 高度分散，无全国性龙头
- 传统零售模式，毛利率通常较高
- 地域性强，物流成本高
- 客户购买频率低，但单次金额高

**NFM 的竞争优势**：
- 超大规模（数百英里范围内最大的家具店）
- 超低价格（利润率仅 4-5%，行业平均 40%+）
- 超高周转（薄利多销）
- 客户口碑极佳，有人千里迢迢赶来购买特价商品

### 收购契机

**出售原因**：
- B 夫人年事已高（89 岁）
- 希望公司长期传承
- 担心后代经营能力
- 信任巴菲特能保持公司文化

**交易过程**：
- 在微笑、握手的基础上完成
- 签署的合同长度仅占一页纸的 1/4
- 巴菲特亲手起草
- 被称为"奥马哈历史性的握手"
- 巴菲特没有检查库存或房地产产权
- 账目都没有经过审计
- 巴菲特有比这更好的洞察力：了解人性，知道他们值得信任

---

## 巴菲特的调研与分析

### 调研方法

**长期观察**：
- 多年前就了解 NFM 和 B 夫人
- 持续关注其经营表现
- 钦佩 B 夫人的经营哲学
- 非常熟悉奥马哈民众在 NFM 的优质购物体验

**实地考察**：
- 参观 NFM 门店
- 观察客流和销售情况
- 与 B 夫人深入交流
- 体验"物美价廉说实话"的经营理念

### 关键发现

1. **难以复制的竞争优势**
   - 规模优势带来采购成本优势
   - 低价策略形成正向循环：低价→高周转→规模更大→更低价格
   - B 夫人的个人品牌是无形资产
   - 客户忠诚度极高

2. **优秀的企业文化**
   - "物美价廉说实话"深入人心
   - 员工忠诚度高
   - 客户口碑极佳
   - 家族精神传承

3. **可持续的商业模式**
   - 家具是刚需，需求稳定
   - 低价策略可持续
   - 规模效应持续增强
   - 地域扩张潜力大

### 核心结论

**投资逻辑**：
- 收购优秀家族企业
- 保持原有管理层和文化
- 长期持有，不干预经营
- 现金流将远超收购价
- B 夫人这样的企业家是稀缺资源

## 投资决策与执行

### 关键时点

| 时间 | 事件 |
|------|------|
| 1983 年初 | B 夫人表达出售意向 |
| 1983 年中 | 巴菲特考察并决定收购 |
| 1983 年 | 完成收购，约 6000 万美元（90%） |
| 1983-1990s | NFM 持续高速增长 |
| 1990s | 收购剩余 10% 股权 |
| 1998 年 | B 夫人去世（104 岁） |
| 2000s | 开设分店，持续扩张 |

### 收购结构

**交易条款**：
- 约 6000 万美元收购 90% 股权
- B 夫人家族保留 10%
- B 夫人继续担任 CEO
- 保持经营独立性
- 合同仅一页纸的 1/4

**伯克希尔的承诺**：
- 不干预日常经营
- 长期持有，不出售
- 保持"物美价廉说实话"理念
- 只提供资本支持

### 收购后的发展

**规模扩张**：
- 1983 年：只有一个营业场地，约 20 万平方英尺
- 现在：营业面积达 168.9 万平方英尺，占地 77 英亩
- 距离巴菲特的家仅两英里
- 经营超过 85000 种商品（家具、电脑、电视、家电等）

**地理拓展**：
- 堪萨斯城开设大型店面
- 达拉斯沃斯堡开设大型店面
- 得梅因开设大型店面
- 40 年只开了三家新店
- 巴菲特和芒格坚持：只有能建立或提升竞争力、具备经济特许经营权的地方才投资
- 不为了追求野心而建立帝国
- 不主观假设在一个地方起作用的东西可以复制到其他地方

**管理层传承**：
- B 夫人工作至 103 岁（每周 70 小时）
- 1998 年 B 夫人去世（104 岁）
- 儿子路易接手
- 另外两个儿子罗纳德和欧文加入
- 现在家族第四代人一起工作

## 投资逻辑分析

### 为什么是优秀家族企业

| 特征 | NFM | 评价 |
|------|-----|------|
| 优秀管理层 | ✅ B 夫人传奇企业家 | 极强 |
| 企业文化 | ✅ 物美价廉说实话 | 强 |
| 竞争优势 | ✅ 规模 + 低价 | 强 |
| 现金流 | ✅ 稳定充沛 | 强 |
| 传承意愿 | ✅ 信任伯克希尔 | 强 |

### 关键洞察

1. **家族企业的独特价值**
   - 创始人精神难以复制
   - 长期视角 vs 短期业绩
   - 文化传承是核心资产
   - B 夫人这样的企业家是稀缺资源

2. **低价策略的护城河**
   - 低价→高周转→规模优势→更低价格
   - 竞争对手难以跟进
   - 客户忠诚度极高
   - 口碑传播效应

3. **收购后的整合原则**
   - 保持原有管理层
   - 不干预经营
   - 只提供资本支持
   - 信任胜过尽职调查

### 竞争优势分析

**NFM 的护城河**：
- 规模优势（采购成本低）
- 品牌声誉（B 夫人传奇）
- 客户忠诚度（口碑传播）
- 地理位置（奥马哈总部）
- 家族精神传承

## 学习要点

### 1. 家族企业的独特价值

创始人精神难以复制，长期视角胜过短期业绩，文化传承是核心资产。B 夫人这样的企业家是稀缺资源，值得信任和支持。

### 2. 低价策略可以成为护城河

薄利多销→规模优势→更低价格的正向循环，竞争对手难以跟进，客户忠诚度极高。

### 3. 收购后不干预经营

优秀管理层 + 自主经营 = 最佳组合。信任胜过尽职调查，保持原有文化和经营理念。

### 4. 企业家精神是无价资产

B 夫人 89 岁仍工作，103 岁才退休，这种精神难以复制。顶尖商学院的毕业生未必是她的竞争对手。

### 5. 长期持有的力量

NFM 持续数十年增长，远超收购价。地理扩张谨慎而稳健，40 年只开三家新店。

### 6. 地域扩张的谨慎原则

只有能建立或提升竞争力、具备经济特许经营权的地方才投资。不为了追求野心而建立帝国。不主观假设在一个地方起作用的东西可以复制到其他地方。
