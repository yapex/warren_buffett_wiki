# 国际冰雪皇后：1997-特许经营权收购

## 一、决策场景还原

### 时间与市场环境

| 项目 | 内容 |
|------|------|
| 收购时间 | 1997年秋季启动，1998年初完成交割 |
| 卖方 | Rudy Luther家族（Rudy Luther于1997年去世后，家族需要资金） |
| 买方 | 伯克希尔·哈撒韦 |
| 市场环境 | 1997年美国经济稳健，消费者对快餐甜品需求稳定 |
| 行业背景 | 特许经营快餐行业处于增长期，品牌整合趋势明显 |

### 公司的背景

| 项目 | 内容 |
|------|------|
| 成立时间 | 1930年代McCullough父子发明软冰激凌，以"皇后"命名 |
| 所有权演变 | 1970年John Mooty等以500万美元收购 |
| 业务模式 | 特许经营为主，自营店仅34家 |
| 业务规模 | 23个国家共5,792家门店 |
| 附属品牌 | 409个Orange Julius门店，43个Karmelkorn门店，190个"甜品中心" |
| 产品线 | 终极汉堡、花生布丁、Dilly Bar等 |

### 巴菲特的位置

1997年，巴菲特执掌伯克希尔·哈撒韦已逾30年，是年全球知名的投资大师。此前的1988-1996年间，巴菲特已先后收购了可口可乐、吉列等消费品牌，积累了丰富的特许经营业务投资经验。巴菲特本人是奥马哈当地Dairy Queen门店的常客，而芒格则是明尼苏达州卡斯湖和贝米吉门店的忠实拥趸。

## 二、关键数据

| 项目 | 内容 |
|------|------|
| 收购价格 | 5.878亿美元 |
| 卖方原始成本 | 500万美元（1970年收购价） |
| 27年增值倍数 | 约118倍 |
| 1998年税后利润 | 3,500万美元 |
| 1998年收入 | 4.2-4.6亿美元 |
| 年资本支出 | 仅900-1,000万美元 |
| 资本回报率（1988-1996） | 37%-81% |
| 门店总数（1997） | 5,792家 |

## 三、巴菲特的决策过程

### 1997年秋季——机会降临

1997年秋季，Rudy Luther去世后，其家族决定出售Dairy Queen。Dick Kiphart致电巴菲特表达了出售意向。

**Buffett原话：**
> "查理和我总算发挥了一点产品方面的专业知识：查理几十年来一直是明尼苏达州卡斯湖和贝米吉的冰雪皇后的常客，而我则是奥马哈门店的忠实拥趸。我们真正做到了'把钱花在了嘴巴上'。"

### 1997年秋季——次日即发要约

巴菲特接到电话的次日便发出了收购要约，展现了其果断的决策风格。

**Buffett原话：**
> "国际冰雪皇后(IDQ)在全球23个国家运营5,792家门店——除了少数几家自营店外，其余全部为加盟店。"

### 1997年致股东信——业务阐述

巴菲特在1997年致股东信中向投资者阐述了这笔投资的逻辑：

**Buffett原话：**
> "冰雪皇后在我们眼中是一个非常容易理解的业务。我们理解终极汉堡、花生布丁或Dilly Bar。我喜欢经营它的人，我喜欢这门生意。"

### 1998年初——完成交割

1998年初，伯克希尔正式完成交割，以5.878亿美元将国际冰雪皇后纳入麾下。

## 四、决策对比点

| 维度 | 传统投资者视角 | 巴菲特视角 |
|------|---------------|-----------|
| 1. 业务理解 | 需要复杂分析模型 | 简单业务：理解产品、理解门店、理解消费者 |
| 2. 增长预期 | 关注门店扩张速度 | 关注现有门店盈利能力和资本回报率 |
| 3. 定价依据 | 参考行业平均估值倍数 | 基于内在价值，忽略短期市场情绪 |
| 4. 卖方动机 | 视为交易障碍 | 理解并利用卖方的流动性需求 |
| 5. 决策速度 | 长时间尽职调查 | 次日即发要约——信心来源于对业务的深刻理解 |
| 6. 时间 horizon | 追求短期收益 | 长期持有，视不确定性为长期买家的朋友 |

### 经验教训
- **关注内在价值而非股价**：巴菲特不受市场情绪左右，以合理价格收购优质资产
- **不要随乐观共识买卖**：在他人恐惧时贪婪，在他人贪婪时恐惧
- **不确定性是长期买家的朋友**：深刻理解业务后，不确定性反而带来买入机会
