# 华盛顿邮报：1973年媒体投资经典

> **核心来源**：巴菲特亲笔信（1975-1994年多次提及）+《巴菲特的第一桶金》

---

## 一、决策场景还原

### 时间与市场环境

1973年，美国股市正处于"漂亮50"泡沫破裂后的崩盘中。1970年代初，投资者对一批大型成长股（如雅芳、宝丽来、施乐）趋之若鹜，推高了它们的市盈率至难以置信的水平——雅芳PE达61倍，宝丽来95倍，施乐46倍。

1973-1974年，市场开始清算：
- 道指从1000点跌至约500点
- 雅芳股价跌86%
- 宝丽来跌91%
- 施乐跌71%

巴菲特后来回忆这段时期，他"就像一个流落在荒岛上的极度缺爱者"——找不到合理价格的股票。这为他在市场恐慌中买入创造了条件。

### 公司危机与机会

**华盛顿邮报的特殊背景**：
- 1933年由尤金·迈耶（凯瑟琳·格雷厄姆的父亲）从破产商人手中买下
- 1971年上市（NYSE），格雷厄姆家族通过A股控制投票权
- 因"五角大楼文件"和"水门事件"报道声名鹊起

**1973年的困境**：
- 尼克松政府对公司持敌意态度
- 可能失去佛罗里达两家电视台牌照——这两家电视台占公司盈利的1/3
- 股价从高位下滑，市值跌至约1亿美元
- 市场先生给出了恐慌价格

### 巴菲特的位置

**资金来源**：1973年初，巴菲特通过所罗门兄弟公司发行20年期债券，募资2000万美元，部分用于偿还旧债，其余用于投资。

**年龄与经验**：43岁，已有20年投资经验，运营伯克希尔约8年。

**个人渊源**：巴菲特12岁时曾送过《华盛顿邮报》，后来他透露："我投资的初始资金有一半来自1940年代给该报送报纸的收入。"

## 二、关键数据

### 投资决策

| 指标 | 数值 |
|------|------|
| **投资时间** | 1973年春末至秋天 |
| **投资金额** | 1060万美元 |
| **买入价格** | 约5.63美元/股（分股调整前） |
| **持股数量** | 467,150股（后分股为1,868,000股） |
| **持股比例** | 9.7%（初始）→ 18%（1999年，因回购） |
| **买入时市值** | 约1亿美元 |
| **巴菲特估值** | 4-5亿美元 |

### 投资结果

| 时间点 | 情况 |
|--------|------|
| 1973年买入 | 市值1亿，巴菲特估值4-5亿 |
| 1973年后 | 跌至8000万，仅为内在价值1/5 |
| 1975年 | 公开表示"计划永久持有" |
| 1981年 | 股价回到巴菲特估值，历时8年 |
| 1985年 | 税后每股经营收益 $10.30 |
| 1987年 | 市值 $323M，成本 $9.7M（约33倍） |
| 1993年 | 每年股利收入 $7M，市值超 $4亿 |
| 2005年 | 格雷厄姆控股出售报纸业务给贝索斯 |
| 2014年 | 伯克希尔大部分持股退出 |

## 三、巴菲特的决策过程

### 1975年：首次公开确认——"计划永久持有"

> "符合这些标准的股票，我们打算长期持有；事实上，我们最大的一笔股权投资是467,150股华盛顿邮报B股，成本约1060万美元，**我们计划永久持有**。"
> 
> ——巴菲特，1975年致股东信

这是巴菲特首次公开阐述他的"永恒持股"理念——不是因为股价涨了要持有，而是因为好企业值得永久拥有。

### 1985年：回顾投资逻辑——"绝佳的例子"

> "我们之前提到过，在过去十年里投资环境已经发生了变化——从一个伟大企业完全不受赏识的环境，转变为它们得到充分认可的环境。**华盛顿邮报公司就是一个绝佳的例子**。"
> 
> ——巴菲特，1985年致股东信

巴菲特在1985年特别股息交易时回顾了这笔投资，强调他在1973年市场极度悲观时买入了一家后来被证明是卓越的企业。

### 1987年：阐述买入时机——"市场先生真是一位极其慷慨的好朋友"

> "不过，持有有价证券的劣势有时可以被一个巨大的优势所抵消：偶尔，股票市场会让我们以极其荒谬的低价买到卓越企业的少数股权——远低于通过协议谈判获得控制权时需要支付的价格。**例如，我们在1973年以每股5.63美元买入华盛顿邮报的股票，1987年该公司的税后每股经营收益为10.30美元**。在这样的案例中，市场先生真是一位极其慷慨的好朋友。"
> 
> ——巴菲特，1987年致股东信

### 1987年：关于管理层——"实在想不出还有比这更好的组合了"

> "在我们三项'有价证券但永久持有'的投资中，这一点根本不成问题：有汤姆·墨菲和丹·伯克在大都会/ABC，比尔·斯奈德和路易·辛普森在盖可保险，**凯瑟琳·格雷厄姆和迪克·西蒙斯在华盛顿邮报，我们实在想不出还有比这更好的组合了**。"
> 
> ——巴菲特，1987年致股东信

### 1991年：关于媒体行业变化——"那种刀枪不入的特许经营权的时代，已经一去不复返了"

> "我们在媒体领域有大量投资——既包括直接拥有的布法罗新闻报，也包括我们在华盛顿邮报和大都会/ABC的持股——而由于行业正在经历的结构性转变，**这些投资的内在价值已经大幅缩水**。"
> 
> ——巴菲特，1991年致股东信

> "大多数媒体资产的经济特征至今仍然远好于美国企业的平均水平。只是**那种刀枪不入的特许经营权和取之不尽的丰裕暴利的时代，已经一去不复返了**。"
> 
> ——巴菲特，1991年致股东信

### 1993年：关于买入时机与风险——"如果有人以极低的价格把整家公司卖给你"

> "按照贝塔理论，一只相对于大盘大幅下跌的股票——就像1973年我们买入华盛顿邮报时那样——在低价位时反而比在高价位时'风险更高'。**如果有人以极低的价格把整家公司卖给你，难道你也会觉得这笔交易风险更高吗？**"
> 
> ——巴菲特，1993年致股东信

巴菲特用这个反问讽刺了传统金融理论对风险的定义，强调内在价值而非股价波动才是衡量风险的尺度。

### 1993年：关于管理层选择——"凯瑟琳将会是一位杰出的管理者"

> "**凯瑟琳·格雷厄姆去年从华盛顿邮报公司董事长的职位上退休**。1973年，我们以大约1,000万美元买入了她公司的股票。如今这笔持股每年为我们带来700万美元的股利收入，市值超过4亿美元。当初买入时，我们知道这家公司的经济前景很好。但同样重要的是，**查理跟我都认定凯瑟琳将会是一位杰出的管理者，并会公正地对待所有股东**。后面这一点尤为关键，因为华盛顿邮报公司有两类股票——我们见过有些管理者会滥用这种结构。"
> 
> ——巴菲特，1993年致股东信

### 1994年：关于个人渊源——"有一半来自1940年代给该报送报纸的收入"

> "我投资的初始资金有一半来自1940年代给该报送报纸的收入。三十年后，该公司上市两年后，伯克希尔买入了一大笔股份。"
> 
> ——巴菲特，1994年致股东信

## 四、决策对比点

> 以下问题没有标准答案，供读者思考

### 1. 信息层面
**实际发生**：巴菲特在12岁时就开始接触华盛顿邮报，对这份报纸有深入了解。他还能接触到格雷厄姆家族成员。

**问题**：你对一家公司的"了解"有多少是来自亲身接触和长期观察，而非研究报告？这种了解如何影响你的投资决策？

### 2. 时机层面
**实际发生**：巴菲特在市场崩盘、股价低迷时买入。他后来形容这是"以极其荒谬的低价"买入。

**问题**：你是主动等待市场恐慌的机会，还是倾向于跟随市场情绪追涨杀跌？"别人恐惧时贪婪"说起来容易，做起来难在哪里？

### 3. 仓位层面
**实际发生**：巴菲特投入了1060万美元，这是当时伯克希尔最大的单笔股权投资，约占净资产的相当大比例。

**问题**：当你极度看好一个投资机会时，你会下多大的注？这个仓位比例是如何确定的？

### 4. 风险层面
**实际发生**：市场理论认为1973年买入"风险更高"，因为股价在下跌。但巴菲特认为低估才是真正的风险。

**问题**：你如何定义"风险"？是股价波动，还是本金永久损失的可能性？这两者有何区别？

### 5. 持有层面
**实际发生**：巴菲特承诺"永久持有"，即使后来媒体行业发生结构性变化也没有改变这一承诺。

**问题**：什么条件下你会改变"永久持有"的决定？行业变化、管理层更换、估值过高——哪个因素对你最重要？

### 6. 核心判断
**实际发生**：巴菲特不仅评估企业价值，还评估管理层质量。他选择凯瑟琳·格雷厄姆，是因为相信她既能经营好企业，也会公正对待所有股东。

**问题**：你在买入一家公司时，是否考虑过"如果我必须持有这家公司10年，我会选择谁当CEO"这个问题？

## 五、后续发展

### 时间线

| 年份 | 事件 |
|------|------|
| 1973 | 巴菲特买入华盛顿邮报467,150股，成本1060万美元 |
| 1974 | 巴菲特加入华盛顿邮报董事会 |
| 1975 | 首次公开表示"计划永久持有" |
| 1979 | 股票分股，持股增至1,868,000股 |
| 1986 | 因担任大都会公司董事，暂时退出董事会 |
| 1991 | 巴菲特指出媒体行业结构性变化，内在价值已大幅缩水 |
| 1993 | 凯瑟琳·格雷厄姆退休，伯克希尔持股每年带来700万美元股利 |
| 1996 | 巴菲特重返华盛顿邮报董事会 |
| 2013 | 贝索斯以2.5亿美元收购《华盛顿邮报》报纸业务 |
| 2014 | 伯克希尔卖出约28%持股，收回大量资金 |

### 巴菲特的回顾

**关于投资回报的来源（1987年）**：
巴菲特在1987年信中总结了投资的"三重上涨"：
1. **商业价值飙升**：公司盈利能力不断增长
2. **每股价值提升**：回购减少股本
3. **估值修复**：折扣缩小，股价涨幅超出每股商业价值涨幅

**关于媒体行业的衰退（1991年）**：
巴菲特坦诚地承认，虽然媒体企业仍然比一般企业强，但"那种刀枪不入的特许经营权的时代"已经结束。他选择继续持有，但明确认识到内在价值已经缩水。

**关于管理层的价值（1993年）**：
巴菲特强调，选择凯瑟琳·格雷厄姆不仅因为她能经营好企业，还因为她"会公正地对待所有股东"——这是双层股权结构下至关重要的品质。
