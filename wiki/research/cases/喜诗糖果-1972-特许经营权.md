# 喜诗糖果：1972年特许经营权认知突破

> **参考来源**：《巴菲特的第一桶金》案例20（格伦·阿诺德著，杨天南译，机械工业出版社2019年版）

---

## 案例概述

1972年，巴菲特通过蓝筹印花公司以2500万美元收购喜诗糖果，这是伯克希尔投资哲学从"烟蒂股"向"特许经营权"转变的标志性案例。喜诗以其强大的品牌、定价权和轻资本特性，为伯克希尔创造了超过19亿美元的累计税前利润（截至2014年），成为"经济商誉"概念的经典教科书。

| 关键数据 | 数值 |
|---------|------|
| **收购时间** | 1972年1月3日 |
| **收购价格** | 2500万美元 |
| **有形净资产** | 800万美元 |
| **溢价（商誉）** | 1700万美元（3.125倍PB） |
| **税后利润（1971）** | 约200万美元 |
| **有形资产回报率** | 25% |
| **连锁店数量（1972）** | 167家 |
| **累计税前利润（至2014）** | 19亿美元 |
| **分红（1971-1991）** | 4.1亿美元 |

---

## 投资背景

### 卖方背景

喜诗糖果由查尔斯·A·喜诗的母亲玛丽·喜诗于1921年在加州帕萨迪纳创立。到1971年：
- **连锁店**：167家，主要集中在加利福尼亚州
- **管理者**：劳伦斯·喜诗（创始人之子）于1969年过世，其兄弟哈里无心管理
- **关键人物**：查克·哈金斯（Chuck Huggins），副总裁，实际运营公司

### 买方背景

1971年的巴菲特：
- 已解散合伙公司，转型为控股型投资
- 控制伯克希尔（29%股权）、多元零售公司（40%股权）、蓝筹印花公司
- 拥有大量浮存金可用于投资，但找不到足够的便宜股票
- 正在从格雷厄姆式投资向品质投资转变

---

## 投资决策过程

### 初次接触

1971年11月，蓝筹印花投资顾问罗伯特·弗拉赫里得知喜诗出售消息，与高管威廉·拉姆齐联系巴菲特。

**巴菲特的第一反应**："价格太贵"，挂了电话。

### 重新评估

弗拉赫里和拉姆齐再次联系时，巴菲特已看过财务报表，态度有所软化，但仍设定严格底线：**2500万美元**。

**关键说服者**：艾拉·马歇尔（惠勒·芒格公司合伙人）努力说服巴菲特和芒格，这是一家与众不同的公司，值得更高价格。

### 会面与尽职调查

巴菲特、芒格、盖林前往洛杉矶与哈里·喜诗和查克·哈金斯会面。

**芒格的优势**：作为加州人，深知喜诗在当地的品牌声誉和品质口碑。

**哈金斯的表现**：坦诚陈述公司优缺点，给巴菲特团队留下深刻印象。巴菲特确认哈金斯可以继续管理公司运营。

### 交易达成

- **初始要价**：3000万美元
- **最终成交价**：2500万美元（哈里·喜诗接受，以便继续过自己想要的日子）
- **交易完成**：1972年1月3日，蓝筹印花收购99%股份（剩余1%于1978年收购）

## 投资逻辑分析

### 为什么不是"格雷厄姆式"投资

| 格雷厄姆标准 | 喜诗状况 | 符合？ |
|-------------|---------|--------|
| 价格低于净流动资产 | 2500万 vs 800万有形净资产 | ❌ 3.125倍PB |
| 低市盈率 | 12.5倍PE（2500万/200万） | △ 不算便宜 |
| 强劲的资产负债表 | 有形资产仅800万 | ❌ 资产轻 |

**巴菲特的突破**：第一次以显著高于有形净资产的价格收购，打破了格雷厄姆的框架。

### 关键洞察：特许经营权

1. **品牌忠诚度**
   - 加州消费者从小吃喜诗糖果长大
   - 粉丝具有极强的品牌忠诚度，近乎狂热
   - 竞争对手夺取市场份额需要付出巨大代价

2. **定价权**
   - 1972年售价：每磅1.85美元
   - 巴菲特和芒格预测：只需提价15美分，税前利润可从400万增至650-700万
   - "没有上限的定价权"——巴菲特后来形容为对着魔镜说"更多"

3. **轻资本特性**
   - 有形资产仅800万，年利润200万（25%回报率）
   - 扩张不需要大量资本投入
   - 利润可用于其他投资

4. **心智份额**
   - 巴菲特称之为"心智份额"而非"市场份额"
   - 品牌在消费者心智情感中的重要性

## 收购后的管理

### 管理层激励

**薪酬方案**：5分钟达成，没有书面协议，持续数十年。

**管理哲学**：
- 不干预日常运营，哈金斯完全自主
- 巴菲特关注财务结果，尤其是资本回报率
- 随时可电话咨询，平等合作而非老板与雇员

### 品质坚持

巴菲特和哈金斯一致同意：
- 只用最好的原料，不添加防腐剂
- 做好客户服务
- 即便短期内影响盈利，也要坚持品质
- 加深、扩宽护城河

### 扩张尝试与教训

**失败尝试**：
- 1980年代在科罗拉多、密苏里、德克萨斯开设分店
- 品牌识别度不足，消费者不愿为陌生品牌支付溢价
- 最终关门

**成功扩张**：
- 1998年推动网上销售策略
- 2012年后谨慎向东部扩展（季节性手推车测试）
- 每家新店投资控制在30万美元以内

**核心教训**：只在顾客喜爱其独特风味并愿意支付溢价的地方开店。

## 定价权的验证

### 数据对比（1972 vs 1982）

| 指标 | 1972年 | 1982年 | 变化 |
|------|--------|--------|------|
| 每磅售价 | $1.85 | $5.10 | +176% |
| 通货膨胀 | - | - | +137% |
| 营业额 | 基数 | 4倍 | +300% |
| 分店数量 | 167家 | 202家 | +21% |
| 单店销售磅数 | 基数 | +18% | - |
| 单店收入 | 基数 | 3倍 | +200% |
| 利润 | 基数 | 5倍多 | +400%+ |

**关键发现**：价格上涨超过通胀，利润增长远超收入增长。

### 长期业绩（1971-2014）

| 时间段 | 关键数据 |
|--------|---------|
| 1971-1991（20年） | 利润增长10倍，分红4.1亿美元 |
| 1972-1999 | 税前利润累计8.57亿美元 |
| 1972-2014 | 税前利润累计19亿美元 |
| 1999年 | 营业利润率24%（食品业极高） |

**哈金斯法则**：巴菲特幽默地指出，哈金斯46岁时税前利润是其年龄的10%（460万），74岁时达到100%（7400万）。

## 经济商誉的概念突破

### 会计商誉 vs 经济商誉

| 维度 | 会计商誉 | 经济商誉 |
|------|---------|---------|
| **定义** | 收购价 - 有形净资产公允价值 | 品牌、声誉等无形资产创造的溢价能力 |
| **摊销** | 40年摊销，每年减少账面利润 | 不摊销，可能随时间增长 |
| **真实性** | 会计处理 | 真实的经济价值 |

### 喜诗的例证

- **会计处理**：1700万商誉分40年摊销，每年42.5万
- **经济现实**：1983年税后利润达1300万（以2000万有形净资产）
- **结论**：经济商誉完全没有减少，反而增长

**巴菲特的洞察**：
> "在考虑一家企业的真实情况时，会计数字往往会造成更多的困惑。我们需要关注企业真实的经济状况，这比那些按照会计准则提供的统计数字更为重要。"

## 投资哲学意义

### 从格雷厄姆到品质投资的转变

```
格雷厄姆阶段              喜诗阶段                后期阶段
   │                        │                      │
   ▼                        ▼                      ▼
烟蒂股投资          →    特许经营权认知      →    可口可乐、苹果
资产负债表          →    经济商誉            →    长期复利
安全边际            →    定价权              →    护城河
```

### 关键影响

**对巴菲特的影响**：
> "与喜诗公司的盈利相比，我们从喜诗赚到的钱远远超过喜诗本身，事实上，它教会了我一些东西，并且，我确信查理也从中受益匪浅。"

**后续投资决策**：
- 1988年投资可口可乐（类似的品牌特许经营权）
- 更加青睐强大的品牌
- 接受为优质企业支付合理溢价

## 学习要点

> 提炼自《巴菲特的第一桶金》案例20

### 1. 在有多个卖家竞争的市场购买原料；在你拥有定价权的市场销售产品

巴菲特说："当你想买的时候，最好是在一个已商品化的市场中采购（有很多卖家出售同类型产品）；当你想卖的时候，最好你的顾客喜欢这个品牌，并愿意为此支付溢价。"

### 2. 总是称赞你的重要经理人

巴菲特总是会提到CEO哈金斯的表现："他对于产品质量和友善服务的狂热坚持，赢得了顾客、员工和股东们的好评。"

### 3. 琢磨客户想法

喜诗品牌在美国西海岸深入人心，它"还有巨大的资产没有在资产负债表上显现出来：宽广的、可持续的竞争优势赋予了它强大的定价权。"

### 4. 追求以小资本取得高回报

以较小的投资资本取得巨大的利润成长，喜诗公司是二者非凡的结合。

### 5. 为特许权出价物超所值

起初收购喜诗的时候，看到价格远远超过公司的有形净资产，巴菲特和芒格有些不太情愿。但是后来回顾，喜诗的高资产回报率反而显得2500万美元的出价是便宜了。

### 6. 持续进化

巴菲特经历了喜诗的非凡成长，使他开始青睐强大的品牌，这也促使他日后做出很多赚大钱的投资。

## 与相关案例的对比

### 与美国运通（1964）的对比

| 维度 | 美国运通（1964） | 喜诗糖果（1972） |
|------|-----------------|-----------------|
| **投资背景** | 危机中的品质投资 | 正常收购优质企业 |
| **特许经营权** | 危机后确认完好 | 一开始就明确认可 |
| **定价权** | 隐含的定价权 | 明确的定价权认知 |
| **投资哲学** | 转变期 | 成熟期 |
| **持有期** | 3年（卖出） | 永久持有 |

### 与华盛顿邮报（1973）的对比

- 同样是危机中的品质投资
- 同样是媒体/消费品牌的特许经营权
- 喜诗是"小而美"的极致体现
- 华盛顿邮报是"大而不能倒"的媒体品牌
