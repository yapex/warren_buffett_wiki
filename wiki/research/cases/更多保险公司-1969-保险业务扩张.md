# 更多保险公司：1969-保险业务扩张

> **参考来源**：《巴菲特的第一桶金》案例 17（格伦·阿诺德著，杨天南译，机械工业出版社 2019 年版）

---

## 案例概述

1968-1969 年，伯克希尔 - 哈撒韦从纺织公司转型为控股公司。通过出售证券投资赚取 500 万美元税后利润，收购了伊利诺伊国民银行，并扩张保险业务（国民赔偿保险、国民火险和水险、盖特韦承保等）。这是伯克希尔"从纺织到保险帝国"转型的关键时期，展示了资本配置的力量。

### 关键数据

| 项目 | 数据 |
|------|------|
| 1965 年市值 | 不到 2,000 万美元 |
| 1965 年净资产 | 2,200 万美元 |
| 1968-1969 年证券盈利 | 税后 500 万美元 |
| 1967 年收购 | 国民赔偿保险（680 万美元） |
| 1969 年收购 | 伊利诺伊国民银行（1,550 万美元） |
| 纺织业务资本 | 约 1,600 万美元 |
| 银行净资产 | 1,700 万美元 |
| 保险公司资本 | 约 1,500 万美元 |
| 银行 + 保险盈利能力 | 每股约 4 美元 |
| 巴菲特持股成本 | 每股 14.86 美元 |

---

## 投资背景

### 市场环境

**1968-1969 年**：
- 股市喧嚣，狂野股票受追捧
- 巴菲特忙于为伯克希尔增添新子公司
- 对股市上其他买家的情绪态度愈加忧心忡忡

### 伯克希尔状况

**转型中的伯克希尔**：
- **1965 年**：
  - 市值不到 2,000 万美元
  - 净资产 2,200 万美元
  - 唯一主业：利润微薄的纺织业务

- **1968-1969 年**：
  - 卖出所有有价证券（主要是股票）
  - 手中现金越来越多
  - 为未来收购控股提供弹药
  - 从证券投资盈利税后 500 万美元

**肯·蔡斯的反思**（1970 年 4 月）：
> "四年之前，公司管理层犯了错误，误以为只要向纺织行业进行大量、持续的资本投入，就可以获得更多、更持久的盈利能力，但实际上这是不可能的。"

> "那些继续投入大量资金用于扩大纺织业务的公司，产生的是'完全不足的回报'。"

---

## 巴菲特的资本配置

### 关键时点

| 时间 | 事件 |
|------|------|
| 1965 年 | 巴菲特控股伯克希尔 |
| 1967 年 | 收购国民赔偿保险（680 万美元） |
| 1967 年 | 收购国民火险和水险公司 |
| 1968-1969 年 | 卖出所有有价证券 |
| 1968-1969 年 | 证券盈利税后 500 万美元 |
| 1969 年 | 收购伊利诺伊国民银行（97.7%） |
| 1969 年 | 收购布莱克印刷公司（100%） |
| 1969 年 | 收购盖特韦承保公司（70%） |

### 资本配置策略

**从纺织到投资**：
1. **限制纺织投资**
   - 巴菲特坚决限制伯克希尔对纺织业务加大投资
   - 纺织业务占用资本约 1,600 万美元
   - 资本回报率不到 5%

2. **证券投资收益**
   - 1968-1969 年卖出所有有价证券
   - 税后盈利 500 万美元
   - 对于当时市值不到 2,000 万美元的公司，这是相当有分量的

3. **收购保险公司**
   - 1967 年：国民赔偿保险（680 万美元）
   - 1967 年：国民火险和水险公司
   - 1969 年：盖特韦承保公司（70%）
   - 保险业务产生承保利润
   - 浮存金成本为零

4. **收购银行**
   - 1969 年：伊利诺伊国民银行（97.7%，1,550 万美元）
   - 银行产生惊人收益
   - 与国民赔偿保险一起构成伯克希尔帝国强大支柱

5. **收购媒体**
   - 1969 年：《奥马哈太阳报》
   - 财务角度不太重要
   - 但为未来媒体投资积累经验

### 保险业务扩张

**国民赔偿保险公司**：
- 1967 年收购
- 资金来源：伯克希尔纺织业务
- 1969 年已可产生承保利润
- 浮存金成本为零

**国民火险和水险公司**：
- 1967 年收购
- 合伙公司旗下
- 浮存金可用于投资

**盖特韦承保公司**：
- 1969 年收购 70%
- 批发经纪商
- 为经纪人提供承销机会
- 至今依然在运营

**再保险业务**：
- 1969 年已开始考虑
- 在洛杉矶建立办公室
- 进入加利福尼亚劳工补偿市场
- 新的再保险部门整装待发
- 计划为新的州提供保险服务

## 投资逻辑分析

### 资本配置原则

| 原则 | 描述 |
|------|------|
| **寻找高资本回报率企业** | 纺织不到 5%，银行保险超过 10% |
| **保持低风险** | 避免大量负债 |
| **成为投资者，而非投机者** | 深入分析企业，追求合理回报 |
| **建立安全边际** | 低价买入优秀企业 |

### 关键洞察

1. **重要原则比宏大计划更为重要**
   - 以合理价格买进低风险的、具有良好资本回报的企业
   - 这是非常重要的原则
   - 可以各种不同的方式应用
   - 构建企业帝国并非一开始就可以预见

2. **内在价值是关注的焦点**
   - 市场或许会在一段时期内将股价推动偏离其内在价值
   - 但最终，价格会反映价值

3. **资本配置是巴菲特投资表现优异的核心**
   - 对于多个行业的估值分析
   - 使得巴菲特和他的管理团队幸免于短视
   - 避免了仅专注单一公司类型所导致的灾难性后果

4. **机会成本思维**
   - 用于衡量另一项替代投资可能取得的最佳回报
   - 如果资金被投资于行业 A 的一家公司
   - 我们需要知道，这笔资金因此就无法被投资到行业 B
   - 如果行业 A 回报 10%，行业 B 预期 9%，应该投资 A

### 巴菲特的重要声明

**1969 年**：
> 巴菲特宣布他希望集中精力于控股公司，因此，蔡斯说他预期伯克希尔 - 哈撒韦不会再"购买更多有价证券"。

**事后看来**：
- 这是个相当有趣的声明
- 伯克希尔后来以股票投资成功而闻名
- 例如购买华盛顿邮报、可口可乐股票而大赚特赚
- 这表明构建企业帝国并非既定宏大计划步步推进的成果
- 而是在不同阶段采取不同策略应用重要原则的结果

## 投资哲学意义

### 在投资哲学演变中的位置

1968-1969 年是伯克希尔**从纺织公司向控股公司转型的关键期**：

- **格雷厄姆元素**：证券投资收益
- **费雪/芒格元素**：控股优秀企业
- **独特贡献**：资本配置，保险浮存金模式
- **后续影响**：保险帝国构建

### 与前后案例的关联

| 关联案例 | 关系 |
|---------|------|
| 国民赔偿保险 -1967 | 保险业务起点 |

| 投资人关系 -1967-1969 | 同期市场背景 |

### 影响因素

- **纺织业务困境**：推动转型
- **保险业务成功**：验证浮存金模式
- **银行收购成功**：展示控股价值
- **资本配置能力**：核心竞争力

## 学习要点

### 1. 重要原则比宏大计划更为重要

以合理价格买进低风险的、具有良好资本回报的企业，这是非常重要的原则，这一原则可以各种不同的方式应用。

### 2. 内在价值是关注的焦点

市场或许会在一段时期内将股价推动偏离其内在价值，但最终，价格会反映价值。

### 3. 资本配置是巴菲特投资表现优异的核心

对于多个行业的估值分析，使得巴菲特和他的管理团队幸免于短视，避免了仅专注单一公司类型所导致的灾难性后果。

### 4. 机会成本思维

资本配置对于公司而言会产生一个机会成本，用于衡量另一项替代投资可能取得的最佳回报。

### 5. 从纺织到保险的转型

伯克希尔的转型并非预先规划的宏大计划，而是在不同阶段应用重要原则的自然结果。
