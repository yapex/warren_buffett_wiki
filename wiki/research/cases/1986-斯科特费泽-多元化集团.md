# 斯科特·费泽：1986-多元化集团收购

> **参考来源**：《巴菲特的伯克希尔崛起》案例 5（格伦·阿诺德著）

---

## 案例概述

1986 年，伯克希尔以 3.15 亿美元收购斯科特·费泽公司 100% 股权。这是一家拥有 20 多个子公司的多元化集团，包括百科全书、吸尘器、工业产品等业务。这是巴菲特"收购优秀企业群"模式的典型代表，展示了如何评估和管理多元化企业。

### 关键数据

| 项目 | 数据 |
|------|------|
| 收购时间 | 1986 年 |
| 收购价格 | 3.15 亿美元 |
| 业务单元 | 20+ 个子公司 |
| 核心业务 | 世界图书百科全书、柯比吸尘器 |
| 1986 年利润 | 约 4000 万美元 |
| 市盈率 | 约 8 倍 |
| 最终结果 | 持续产生稳定现金流 |

---

## 投资背景

### 斯科特·费泽公司背景

**公司业务**：
- 世界图书百科全书（市场领导者）
- 柯比吸尘器（直销模式）
- 工业产品（多个细分市场）
- 金融服务

**公司状况**：
- 多元化经营，业务分散
- 部分业务表现优秀
- 管理层希望出售
- 估值合理

### 收购契机

**出售原因**：
- 公司被敌意收购威胁
- 管理层寻求友好买家
- 希望保持公司完整
- 信任巴菲特能妥善经营

**巴菲特的兴趣**：
- 估值合理（约 8 倍 PE）
- 部分业务具有经济特许权
- 管理层优秀
- 符合伯克希尔收购标准

---

## 巴菲特的调研与分析

### 调研方法

**业务分析**：
- 逐一评估 20+ 个业务单元
- 识别具有经济特许权的业务
- 分析各业务的竞争地位

**管理层评估**：
- 与 CEO 多次交流
- 评估管理层能力
- 确认留任意愿

### 关键发现

1. **部分业务具有经济特许权**
   - 世界图书百科全书：市场领导者
   - 柯比吸尘器：品牌优势
   - 其他业务：多数在细分市场领先

2. **估值合理**
   - 3.15 亿美元收购价
   - 约 8 倍市盈率
   - 低于内在价值

3. **管理层优秀**
   - CEO 能力强
   - 愿意继续经营
   - 认同伯克希尔理念

### 核心结论

**投资逻辑**：
- 收购价格低于内在价值
- 部分业务具有经济特许权
- 管理层优秀，愿意留任
- 现金流将远超收购价

## 投资决策与执行

### 关键时点

| 时间 | 事件 |
|------|------|
| 1985 年底 | 斯科特·费泽寻求买家 |
| 1986 年初 | 巴菲特开始调研 |
| 1986 年 1 月 | 达成收购协议 |
| 1986 年 | 完成收购，3.15 亿美元 |
| 1986-2000s | 持续产生稳定现金流 |

### 收购结构

**交易条款**：
- 3.15 亿美元收购 100% 股权
- 管理层留任
- 保持业务完整
- 伯克希尔不干预经营

**收购后管理**：
- 各业务单元独立运营
- CEO 向巴菲特汇报
- 资本配置由伯克希尔决定

## 投资逻辑分析

### 为什么收购斯科特·费泽

| 因素 | 说明 |
|------|------|
| **估值** | 8 倍 PE，低于内在价值 |
| **业务质量** | 部分业务具有经济特许权 |
| **管理层** | 优秀，愿意留任 |
| **现金流** | 稳定，可预测 |
| **协同效应** | 可纳入伯克希尔体系 |

### 关键洞察

1. **多元化企业的评估方法**
   - 逐一评估各业务单元
   - 识别核心竞争优势
   - 加总各部分价值

2. **合理价格收购企业群**
   - 不是每个业务都优秀
   - 但整体估值合理
   - 现金流充足

3. **收购后的整合原则**
   - 保持业务完整
   - 管理层留任
   - 不干预日常经营

### 竞争优势分析

**斯科特·费泽的护城河**：
- 世界图书百科全书：品牌优势
- 柯比吸尘器：直销网络
- 工业产品：细分市场领导地位

---

## 学习要点

### 1. 多元化企业的评估

逐一分析各业务单元，识别核心竞争优势。加总各部分价值，评估整体估值。

### 2. 合理价格收购企业群

整体估值合理即可，不必每个业务都完美。现金流充足是关键。

### 3. 收购后保持完整

不拆分，不重组，保持原有运营。各业务单元独立运营。

### 4. 管理层留任是关键

优秀管理层继续经营，确保业务稳定。CEO 向巴菲特汇报，资本配置由伯克希尔决定。

### 5. 现金流导向

关注现金流而非会计利润。稳定现金流就是好资产。
