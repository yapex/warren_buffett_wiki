# 吉列/宝洁：1989-消费品牌投资

> **参考来源**：《巴菲特的伯克希尔崛起》案例 10（格伦·阿诺德著）

---

## 案例概述

1989 年，伯克希尔投资 6 亿美元购买吉列公司可转换优先股。这是巴菲特对消费品牌的又一次重注，与可口可乐投资理念一致。2005 年吉列被宝洁收购后，伯克希尔转换为宝洁股票，继续持有。这笔投资展示了"无法回避的公司"和"剃须刀 + 刀片"商业模式的价值，最终回报超过 10 倍。

### 关键数据

| 项目 | 数据 |
|------|------|
| 投资时间 | 1989 年 |
| 投资金额 | 6 亿美元 |
| 投资形式 | 可转换优先股 |
| 股息率 | 8.75% |
| 转换价格 | 约定价格 |
| 2005 年 | 吉列被宝洁收购（570 亿美元） |
| 转换后 | 持有宝洁股票 |
| 最终回报 | 超过 10 倍 |

---

## 投资背景

### 吉列公司背景

**公司地位**：
- 全球剃须刀市场领导者
- 市场份额超过 60%
- "剃须刀 + 刀片"商业模式典范
- 品牌价值全球领先

**商业模式**：
- 剃须刀架低价销售
- 刀片高利润重复购买
- 客户锁定效应强
- 持续现金流
- 男性必需品，需求稳定

### 投资契机

**吉列的需求**：
- 需要资本金支持扩张
- 防御敌意收购
- 增强资产负债表

**巴菲特的考虑**：
- "无法回避的公司"
- 商业模式优秀
- 可转换优先股提供安全边际
- 8.75% 股息率有吸引力

---

## 巴菲特的调研与分析

### 调研方法

**商业理解**：
- 深刻理解"剃须刀 + 刀片"模式
- 了解吉列的市场地位
- 评估竞争优势

**行业分析**：
- 剃须刀市场稳定增长
- 吉列主导地位稳固
- 新进入者威胁小

### 关键发现

1. **"无法回避的公司"**
   - 男性必需品
   - 市场份额 60%+
   - 品牌忠诚度极高
   - 定价权强

2. **优秀的商业模式**
   - "剃须刀 + 刀片"锁定客户
   - 刀片重复购买
   - 持续现金流
   - 高利润率

3. **可转换优先股的优势**
   - 8.75% 固定股息
   - 优先清偿权
   - 转换权提供上行潜力

### 核心结论

**投资逻辑**：
- 最强消费品牌之一
- 商业模式优秀
- 可转换优先股提供安全边际
- 长期看好

## 投资逻辑分析

### 为什么投资吉列

| 因素 | 说明 |
|------|------|
| **市场地位** | 全球领导者，60%+ 份额 |
| **商业模式** | "剃须刀 + 刀片"，锁定客户 |
| **品牌价值** | 全球最强消费品牌之一 |
| **投资工具** | 可转换优先股，安全边际 |
| **股息率** | 8.75%，有吸引力 |

### 关键洞察

1. **"剃须刀 + 刀片"模式的价值**
   - 初始产品低价
   - 消耗品高利润
   - 客户锁定效应
   - 持续现金流

2. **消费必需品的抗周期性**
   - 剃须是男性必需品
   - 经济周期影响小
   - 需求稳定

3. **可转换优先股的灵活性**
   - 下行保护
   - 固定收益
   - 上行潜力
   - 2005 年转换为普通股

### 竞争优势分析

**吉列的护城河**：
- 市场份额（60%+）
- 品牌价值（全球领先）
- 专利保护（刀片技术）
- 分销网络（全球覆盖）

## 学习要点

### 1. "无法回避的公司"

寻找那些消费者必须购买的公司。剃须是男性必需品，经济周期影响小，需求稳定。

### 2. "剃须刀 + 刀片"模式

初始产品 + 消耗品的商业模式价值。客户锁定效应强，持续现金流。

### 3. 可转换优先股的价值

在不确定时提供安全边际。下行保护、固定收益、上行潜力。

### 4. 消费必需品的抗周期性

经济周期影响小，需求稳定。剃须是男性必需品。

### 5. 并购事件的处理

2005 年吉列被收购，灵活转换为宝洁股票。继续持有，享受增长。
