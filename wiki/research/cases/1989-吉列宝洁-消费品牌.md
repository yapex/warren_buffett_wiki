# 吉列/宝洁：1989-消费品牌投资

> **核心主题**："无法回避的公司"和"剃须刀+刀片"商业模式的价值
> **时间跨度**：1989-2005 年
> **关键人物**：科曼·雷夫利（吉列 CEO）

---

## 一、决策场景还原

### 巴菲特的位置

1989 年，巴菲特刚刚完成可口可乐投资，正在继续寻找消费品牌投资机会。此时他：
- 已形成明确的品质投资理念
- 偏好"无法回避的公司"
- 善于运用可转换优先股工具

### 吉列公司背景

**市场地位**：
- 全球剃须刀市场领导者
- 市场份额超过 60%
- "剃须刀 + 刀片"商业模式典范

**商业模式**：
- 剃须刀架低价销售
- 刀片高利润重复购买
- 客户锁定效应强
- 持续现金流
- 男性必需品，需求稳定

### 投资契机

**吉列的需求**：
- 需要资本金支持扩张
- 防御敌意收购
- 增强资产负债表

## 二、关键数据

### 投资决策数据

| 项目 | 数据 |
|------|------|
| 投资时间 | 1989 年 |
| 投资金额 | 6 亿美元 |
| 投资形式 | 可转换优先股 |
| 股息率 | 8.75% |
| 转换价格 | 约定价格 |
| 优先清偿权 | 有 |

### 投资结果数据

| 项目 | 数据 |
|------|------|
| 2005 年 | 吉列被宝洁收购（570 亿美元） |
| 转换后 | 持有宝洁股票 |
| 最终回报 | 超过 10 倍 |

---

## 三、巴菲特的决策过程

### 投资逻辑

**为什么投资吉列**：

| 因素 | 说明 |
|------|------|
| **"无法回避的公司"** | 男性必需品，剃须刀每天要用 |
| **市场地位** | 全球领导者，60%+ 份额 |
| **商业模式** | "剃须刀 + 刀片"，锁定客户 |
| **品牌价值** | 全球最强消费品牌之一 |
| **投资工具** | 可转换优先股，安全边际 |
| **股息率** | 8.75%，有吸引力 |

### 可转换优先股的优势

**投资结构**：
- 8.75% 固定股息 → 下行保护
- 优先清偿权 → 安全边际
- 转换权 → 上行潜力
- 灵活选择 → 2005 年转换为普通股

### 2005 年宝洁收购

**收购条款**：
- 宝洁 570 亿美元收购吉列
- 伯克希尔转换为宝洁股票
- 继续持有，享受宝洁增长

## 四、决策对比点

### 1. "无法回避的公司"：你如何定义？

巴菲特寻找消费者"无法回避"的产品或服务。

**问题**：什么样的公司可以被定义为"无法回避"？这与"护城河"有什么区别？

### 2. 商业模式：什么样的模式最有价值？

"剃须刀 + 刀片"模式的核心是初始产品低价切入，刀片高利润重复购买。

**问题**：这种模式的本质是什么？哪些行业可以应用这种模式？

### 3. 可转换优先股：什么时候使用？

巴菲特使用可转换优先股在不确定时提供安全边际。

**问题**：可转换优先股相比普通股有什么优势？什么时候适合使用这种投资工具？

### 4. 并购应对：你会如何处理？

2005 年宝洁收购吉列时，巴菲特灵活转换为宝洁股票。

**问题**：当你投资的公司被收购时，你会如何决策？现金、股票、还是其他选择？

### 5. 长期持有：你能持有多久？

从 1989 年到 2005 年是 16 年，加上后续持有时间更长。

**问题**：你能接受 16 年以上的投资周期吗？什么因素让你继续持有？

### 6. 行业选择：消费品牌 vs 其他

巴菲特偏好消费品牌，因为需求稳定、抗周期。

**问题**：消费品牌与其他行业相比有什么优势？什么时候应该选择其他行业？
