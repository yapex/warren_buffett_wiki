# 真正的指数化 / True Indexing

## 概念解析 / Definition

> [!zh] 🏳️
> 只有股本回报率上升，即不需追加资本收益就能与物价水平同步增长，才是真正的抗通胀。

> [!en] 🇺🇸
> Only rising ROE where earnings grow proportionally with prices without additional capital constitutes true inflation hedging.

## 核心要义 / Key Principles

在通货膨胀环境下，名义回报必须与物价水平同步增长才能维持实际购买力。巴菲特指出，真正的"资本指数化"需要股本回报率持续上升，而非仅仅依靠追加资本实现名义增长。

## 实践应用 / Practical Application

> [!zh] 🇨🇳
> 要让资本真正实现指数化增长，股本回报率必须上升。

> [!en] 🇺🇸
> For capital to be truly indexed, return on equity must rise.

在高通胀时期，大多数企业需要不断追加资本才能维持经营规模，但这并非真正的增长。真正的指数化要求企业能够以现有资本产生更高回报。
