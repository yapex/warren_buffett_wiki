# 1985年信摘要 / 1985 Letter Summary

## 核心要点

### 1. 财务表现与规模困境
1985年伯克希尔净资产增长6.136亿美元，每股增长48.2%。过去21年（1965-1985）每股账面价值从19.46美元增长到1,643.71美元，年复合增长率23.2%。巴菲特坦诚表示这一增长率未来难以重复，原因：
- **市场规模变化**：当前市场很少有显著低估的股票可供买入
- **规模障碍**：资本规模是十年前的20多倍，"规模的增长终将拖累卓越的经济回报"

> [!quote]
> "要达到15%的平均回报率，需要57亿美元的利润——而过去十年只有约15家美国企业（非石油公司）赚到了这个数字。"

### 2. 纺织业务的关闭
7月决定关闭运营了21年的纺织业务。这段历史充满教训：
- 1965年收购时账面净资产2200万美元，但内在价值更低——之前9年累计销售5.3亿美元却亏损1000万美元
- 管理出色但经济无解：无论多优秀的管理都无法克服大宗商品业务的结构性困境
- 最终教训：**"当你在一条长期漏水的船上时，换船比补漏更有效"**

> [!quote]
> "一匹会数到十的马是一匹了不起的马——但不是一个了不起的数学家。同样地，一家能出色配置资本的纺织公司是一家了不起的纺织公司——但不是一家了不起的企业。"

### 3. 三项优秀业务
内布拉斯加家具商场、喜诗糖果和布法罗晚报继续表现卓越：
- **15年前**合计税前利润约800万美元
- **1985年**合计税前利润达7200万美元
- 仅需额外投入4000万美元资本——展现了经济商誉的强大威力

| 业务 | 1985年税前利润 | 15年前利润 | 备注 |
|------|---------------|-----------|------|
| 内布拉斯加家具商场 | - | - | B夫人92岁仍在全职工作 |
| 喜诗糖果 | - | - | 同店销量略有下滑 |
| 布法罗晚报 | - | - | 家庭渗透率极高 |

### 4. 激励机制反思
巴菲特批评传统的十年期固定价格股票期权制度：
- **留存收益自动创造价值**：期权持有者无需付出就能从留存收益中获益
- **双重标准**：外部人永远不会获得这样的期权
- 伯克希尔的做法：根据各业务单元的实际业绩奖励经理人，与股价无关

> [!quote]
> "没有任何所有者能免除资本成本的负担，而固定价格期权的持有者完全不必承担资本成本。"

### 5. 保险业务
1985年行业综合比率高达118%（1972年以来最差）。问题根源：
- **社会性通胀**：陪审团和法官越来越倾向于让保险公司"大口袋"赔付
- **长尾业务的隐患**：损失准备金估算严重不足

伯克希尔的优势：
- 强大的资本实力使我们在市场紧张时获得竞争优势
- 国民保险公司6月刊登广告宣布承保大额风险（最低保费100万美元），获得600份回复，约5000万美元保费

### 6. 消防员基金份额分担合约
9月1日与消防员基金保险公司(FFIC)签订7%份额分担合约：
- 合同期4年
- 承担FFIC全部有效业务7%的损失和费用
- 获得约9270万美元未到期保费准备金
- 支付2940万美元补偿FFIC已发生的承保费用
- 杰克·伯恩从盖可保险转任FFIC董事长兼CEO

### 7. 重大投资：大都会/ABC
年底以每股172.50美元购买300万股大都会/ABC：
- 投资约5.175亿美元（当时最大的单笔投资）
- 帮助Capital Cities融资35亿美元收购ABC
- 与汤姆·墨菲和丹·伯克的管理团队结盟
- 签订特殊协议：管理层代表巴菲特投票所持股票
- 限制出售权利，确保不会被出售给未经管理层同意的大股东

> [!quote]
> "汤姆·墨菲和丹·伯克不仅是伟大的管理者，更是那种你会想把女儿嫁给他的人。"

### 8. 收购斯科特费泽公司
年底后以约3.2亿美元收购斯科特费泽公司：
- 17项业务，年销售额约7亿美元
- **世界图书公司**(World Book)约占销售额40%，是百科全书行业遥遥领先者
- CEO拉尔夫·谢伊任职9年，表现卓越
- 收购历程有趣：从ESOP计划失败到直接谈判达成
