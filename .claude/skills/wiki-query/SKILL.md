## 功能

### 1. 全文搜索

搜索 Wiki 中的所有段落：
- 支持关键词搜索
- 段落级检索
- 匹配度排序

### 2. 概念时间线

查询特定概念的时间线：
- 显示概念在不同年份的提及
- 按时间顺序排列
- 追踪概念演变

### 3. 文档内搜索

在特定文档中搜索：
- 信件内搜索
- 案例研究内搜索
- 概念页面内搜索

### 4. 引用时间线

查询特定引用的时间线：
- 名言语录的时间分布
- 概念的引用演变

## 使用方法

### 基础查询

```
查询 "安全边际"

```

### 概念时间线

```
查询 "安全边际" 的时间线
搜索 "能力圈" 演变
```

### 文档内搜索

```
在 1985-letter.md 中搜索 "纺织"
在 美国运通案例 中搜索 "危机"
```

### 高级查询

```
查询 "保险 浮存金" --limit=10
搜索 "格雷厄姆" --year=1965-1970
```

## 查询参数

### --limit

限制返回结果数量：
```
查询 "巴菲特" --limit=20
```

### --year

限定年份范围：
```
搜索 "收购" --year=1970-1980
```

### --type

限定文档类型：
```
查询 "保险" --type=letters
查询 "保险" --type=companies
查询 "保险" --type=concepts
```

### --verbose

显示详细信息：
```
查询 "安全边际" --verbose
```

## 搜索技巧

### 1. 使用关键词组合

```
# 好
查询 "安全边际 价格 价值"

# 避免过于宽泛
查询 "投资"  # 结果太多
```

### 2. 使用年份范围

```
# 查找特定时期
搜索 "纺织" --year=1965-1985
```

### 3. 结合概念时间线

```
# 追踪概念演变
查询 "护城河" 的时间线
```

### 4. 在特定文档类型中搜索

```
# 只在信件中搜索
查询 "收购" --type=letters
```
